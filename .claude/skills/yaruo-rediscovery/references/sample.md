# 必読サンプル ── 一つの概念を再発見する

抜粋元：`docs/books/general-relativity.md`「4-2　曲がった空間でベクトルを運ぶ ── 平行移動と接続」

この抜粋では、素朴な成分差分を極座標の反例で壊し、失敗の原因を言語化し、必要な制約から接続を組み立て、具体的な成分を計算し、次の未解決課題へつなげている。内容や固有の比喩を流用せず、再発見の順序、数式を会話で追う密度、失敗の回収を手本にする。

### 4-2　曲がった空間でベクトルを運ぶ ── 平行移動と接続

---

**やらない夫**：
ベクトル場の変化を調べたいとする。
点Pのベクトルと、隣の点Qのベクトルの成分同士を引き算すればいい──と思うか？

**やる夫**：
それでいいんじゃないかお。$dV^\mu = V^\mu(Q) - V^\mu(P)$ で終わりだお。

**やらない夫**：
では反例。平面全体に、真北を向いた一様な風が吹いているとする。
風速 w、どこでも完全に同じ向き・同じ強さの、絶対に変化しないベクトル場だ。
これを **極座標** の成分 $(V^r, V^\theta)$ で見る。

x軸上の点（θ = 0 の方向）では、北はどっち向きだ？

**やる夫**：
x軸上から見ると、北（+y方向）は θ が増える向き、つまり **純粋に θ 方向** だお。
$V^r = 0$ で $V^\theta$ だけが値を持つお。

**やらない夫**：
y軸上の点（θ = 90°）では？

**やる夫**：
y軸上では、北は原点から遠ざかる向き、つまり **純粋に r 方向** だお。
$V^r = w$、$V^\theta = 0$。

……あれ？？ おかしいお。
**風は1ミリも変化してないのに、成分は場所ごとに全然違う**。
x軸上では (0, 何か)、y軸上では (w, 0)。
成分の引き算 $dV^\mu$ はゼロにならないお！

**やらない夫**：
これが罠だ。成分の変化には2つの原因が混ざる：
(a) ベクトル自身が本当に変化した分
(b) 基底ベクトル（座標の目盛りの向き）が場所ごとに回転している分

極座標の基底 $e_r, e_\theta$ は場所ごとに向きが違うから、
不変なベクトルでも成分は変わってしまう。
お前が測りたいのは (a) だけだ。(b) を差し引く仕組みが要る。

**やる夫**：
もっと根本的な問題もある気がしてきたお。
球面みたいに本当に曲がった空間だと、
点Pの接平面と点Qの接平面は傾きが違う別の平面だお。
**そもそも別の場所のベクトル同士は、直接引き算できない**。

**やらない夫**：
その通り。引き算するには、Pのベクトルを何らかの規則でQまで『向きを保ったまま運ぶ』
必要がある。これを **平行移動** と呼ぶ。
ベクトル $V^\mu$ を微小変位 $dx^\alpha$ だけ平行移動したときの成分の補正を、
どういう形に書くべきだと思う？

**やる夫**：
うーん、一般には $V$ と $dx$ の複雑な関数……

**やらない夫**：
制約をかけろ。第一に、変位は微小だ。第二に、
2つのベクトルの和を運んだ結果は、別々に運んで足した結果と同じであるべきだ
（そうでないと結果が基底の取り方に依存してしまう）。

**やる夫**：
微小変位なら $dx$ の高次項は落ちるから $dx^\alpha$ に線形。
和の条件から $V^\beta$ にも線形。
つまり両方に線形で、比例係数は場所ごとの関数だから

$$\delta V^\mu = -\Gamma^\mu{}_{\alpha\beta}\,V^\beta\,dx^\alpha$$

係数の塊 $\Gamma$ が出てきたお。マイナスは補正の向きの約束だお。

**やらない夫**：
$\Gamma^\mu{}_{\alpha\beta}$ を **接続係数（クリストッフェル記号）** と呼ぶ。
各点での『運び方の規則』だ。
ただしこのままでは $\Gamma$ は何でもありだ。物理的に自然な要請を2つ置く。

**要請1（計量整合性）**：平行移動でベクトルの長さと、2つのベクトルの間の角度が変わらない。

**要請2（捩れなし）**：$\Gamma^\mu{}_{\alpha\beta} = \Gamma^\mu{}_{\beta\alpha}$。
微小な平行四辺形がきちんと閉じる、という要請だ（今は深入りしない）。

**やる夫**：
要請1は当然に見えるけど、置かなかったらどうなるんだお。

**やらない夫**：
1メートルの棒を平行移動して持ち帰ったら80cmになっている世界を想像しろ。
『長さ』が移動の履歴に依存して、そもそも計量 $g_{\mu\nu}$ で幾何を定義した意味が壊れる。
運搬と物差しが喧嘩しないための整合条件だ。

さて、この2つの要請を課すと、驚くべきことに
$\Gamma$ は **計量から一意に決まってしまう**

$$\Gamma^\mu{}_{\alpha\beta} = \frac{1}{2}\,g^{\mu\sigma}\!\left(\partial_\alpha g_{\beta\sigma} + \partial_\beta g_{\alpha\sigma} - \partial_\sigma g_{\alpha\beta}\right)$$

**やる夫**：
$g^{\mu\sigma}$ って添字が上についてるのは何だお。

**やらない夫**：
$g_{\mu\sigma}$ の逆行列だ。添字の上げ下げに使う。今は逆行列とだけ思っておけ。

式を眺めてるだけでは身につかない。極座標（$g_{rr}=1,\ g_{\theta\theta}=r^2$）で
$\Gamma^r{}_{\theta\theta}$ を計算してみろ。

**やる夫**：
やってみるお。$\mu = r$、$\alpha = \beta = \theta$ で、$g^{rr} = 1$ だから

$$\Gamma^r{}_{\theta\theta} = \frac{1}{2}\,g^{rr}\left(\partial_\theta g_{\theta r} + \partial_\theta g_{\theta r} - \partial_r g_{\theta\theta}\right) = \frac{1}{2}(0 + 0 - 2r) = -r$$

$\Gamma^r{}_{\theta\theta} = -r$ だお。同じように計算すると
$\Gamma^\theta{}_{r\theta} = \Gamma^\theta{}_{\theta r} = 1/r$ で、残りは全部ゼロだお。

……でも待つお。**極座標はただの平面** だお？
平面なのに $\Gamma \neq 0$ ってことは、
$\Gamma$ も『座標の歪み』と『空間の歪み』を混ぜて表示してるんだお。

**やらない夫**：
その通りで、実は $\Gamma$ は座標変換でテンソルとして変換しない──
平坦な空間ですら、座標の選び方次第でゼロにも非ゼロにもなる。
だが混ぜたままでも、まず先にできる仕事がある。『微分の修理』だ。
