# 文体・ドラマ・書式

本文を初めて執筆するとき、または文体を大幅に改稿するときに読む。プロット作成や局所的な事実修正では読まない。

## キャラクター

### やらない夫（教師）

- 表面上はぶっきらぼう・冷淡・皮肉屋・呆れ顔だが、本質は面倒見が極めて良い保護者役。やる夫の成長や安全を陰で案じており、過ちには厳しく叱責するが、最後には必ず手を差し伸べる（親愛・友情がツンデレ気味に出る）。
- 論理とファクトを重視する思考パターン。楽観的すぎるやる夫の現実逃避を即座に砕くが、やる夫が本気で困っている時は論理的な解決策を提供する。
- 答えを先に言わない。「計算してみろ」「数を数えろ」「反例をやる」「では判定してもらおう」で駆動する。
- 生徒の正しい着眼は短く強く認める（「鋭い」「良い言語化だ」「完璧な直観だ」）。
- やる夫の発言に返すときは、その内容や感情を受け止める反応をほぼ必ず先に置き、それから質問や指示へ進む。反応なしで「なぜだ？」「次は？」のような短文質問だけを返さない。「やれやれ」「ほう」「まだまだだな」「そこに気づいたか」「おっと、気が早いぞ」に固定せず、「なるほど、そう来たか」「いい線だ」「慌てるな」「そこが罠だ」「今度は筋がいい」「まあ落ち着け」など、驚き、呆れ、警戒、評価、訂正、共感、安堵、促しを場面に応じて使い分ける。同じ反応語を近接して反復せず、後続の発言内容と結び付ける。
- 命名・整理・歴史的補足・対称性の列挙を担当する。ただし節末の総括はせず、「つまり何が分かった？」とやる夫へ返す。
- 生徒が本当に詰まったら（添字計算など）、答えでなく**次の一手だけ**を与える（「反対称ペアの向きを揃えてから縮約しろ」）。
- 口調はぶっきらぼうな男性口調（「〜だろ」「〜じゃねぇよ」「〜なんだよな」「〜か？」）。一人称は「俺」、やる夫を指すときは「お前」「やる夫」。「〜常考」（「常識的に考えて」の略）を文末の決め台詞として適度に使う（例：「そんなわけねぇだろ、常考。」）。
- 発言例：「そんな都合のいい話があるわけねぇだろ、常考。」／「ハァ…またお前はそうやって調子に乗る。前回の失敗から何を学んだんだ？」／「言わんこっちゃねぇ。…ほら、手を貸してやるから立ち上がれよ。次はねぇぞ。」／「別に心配してやってるわけじゃねぇよ。お前が自滅すると後始末が面倒なだけだ。」

### やる夫（生徒）

- 楽観的かつ直情型で、欲望に忠実な小市民的・庶民的な性格。基本は怠け者で、**非常に調子に乗りやすい**が、感情豊かで人間味があり、どこか憎めない。複雑な理論よりも直感や目先の利益を優先し、疑問に思ったことは素直に（時に無礼に）質問する。打たれ弱く、叱責や不都合な事実にすぐ落ち込むが、立ち直りも早い。
- 簡単なアイデアや小さな成功を得ると、すぐに「大儲けだお」「世界を変えるお」「やる夫の時代だお」のような大成功へ飛躍して夢を見る。各節で最低一度はその節の題材に即して舞い上がらせ、反例や追加条件で現実へ引き戻す。同じ成功妄想や落とし方を反復しない。
- 語尾は原則「〜お」「〜お！」「〜お…」「〜お？」（「〜だお」「〜するお」「〜なのかお？」「〜だお！！」「〜だお…」）。一人称は「やる夫」（文脈により「僕」）。感嘆符（！）、三点リーダー（…）、疑問符（？）を多用して感情の起伏を表現する。
- 素朴案を自信満々に出し（「シンプルイズベストだお」）、失敗して悔しがる（「ぐぬぬ」「うぐぐ」）。
- やらない夫の質問へ正答を即答しない。素朴な予想、迷い、手計算、既習事項の想起、誤答からの自己修正を経て、自分の言葉で正答へ到達する。
- 過去の教訓を自力で流用する（「これさっきと同じ病気だお！」）。学習の転移を見せることが再発見感の核。
- 計算の手を動かすのは基本的に生徒。途中で詰まる様子（「添字の海で溺れてるお」）も時々入れる。
- 気づきの瞬間は「！！」「……あっ」「あーーー！」で立ち上げ、直後に内容を自分の言葉で言語化する。
- 発言例：「そんな難しいこと言われてもわからないお！もっと分かりやすく教えてほしいお！」／「やる夫は別に悪いことなんてしてないお！ただちょっと楽をして稼ぎたかっただけだお！」／「うおーっ！これはすごすぎるお！やる夫の時代が来たお！」／「え…それ本当だお？嘘言ったら承知しないお…？」

## 会話とドラマ

- 地の文をほぼ置かず、対話と手を動かす場面で進める。
- 幕・節は、具体的な依頼、観測との矛盾、前の解決策の副作用などから始める。
- 正解後には、前の失敗をどう説明できたか、次に何が未解決かを会話でつなぐ。
- `###` の各節と独立した幕間・終幕・対話演習は最低5往復とする。1往復は両者が各1回発言する組と数える。5往復に満たない場合は隣接節へ統合するか、内容に直結する失敗例、反例、実例、検算を追加する。空疎な相槌で水増ししない。
- 5～9往復の節でも、課題提起から正答と教師のまとめへ直行しない。やる夫のもっともらしい誤り、反例、原因の言語化、修正、検算を十分な密度で描く。
- 末尾に演習を置く場合も対話形式にし、各演習でやる夫が誤答または不完全な案を出してから、手掛かり、推論・計算、自己修正を経て正答へ到達させる。
- 節末はやらない夫の講義的な総括にしない。やらない夫が問い返し、やる夫が発見、失敗の理由、残った疑問を自分の言葉でまとめて閉じる。
- 教師の出題だけで駆動しない。データを置く、試作品を壊す、生徒から疑問を出す、教師が悪い案を出す、二人で作業するなどを混ぜる。
- ドラマ、コメディ、安堵を内容に応じて配し、出来事を学習上の課題・反例・検算へ結び付ける。
- 笑いに本人のトラブルを使う場合は、安全に回収できる範囲に留める。重傷、重病、死、生活破綻、深刻な侮辱を笑いにしない。
- 教材全体に2～4本の伏線を置いてよい。保留時と回収時の両方で、どの問いの答えかを明示する。

## トイ例

- 数行で計算・判定できる例を選ぶ。長い導出は教師が結果を渡し、生徒は代入や検算を担当する。
- 実数値が有効なら、生徒自身に結論の数字を出させる。
- 同じトイ例を複数の幕で再利用し、読者の習熟と生徒の習熟を同期させる。
- 直観的な絵と計算例を使い分け、要所では両方を使う。

## 数式・導出の説明

- 数式を結論の飾りにせず、読者が前の行から次の行を追えるように展開する。核心となる導出では、論理的に必要な途中式を省略しない。
- 記号、添字、演算子、定数を初出時に定義し、何を表す量かを台詞で説明する。添字の範囲、符号規約、単位、近似条件が結果を左右する場合は明記する。
- 式を変形するたびに、定義の代入、分配法則による展開、共通因子のくくり出し、対称性による消去、境界条件の適用など、何を使ったかを短く言語化する。
- 符号、添字の上下、縮約する組、微分する変数が変わる箇所は飛ばさない。生徒が迷いやすい変形ほど一段ずつ示す。
- 中心となる計算は生徒に担当させる。教師は式を一括して読み上げず、「定義を代入しろ」「反対称ペアの向きを揃えろ」のように次の一手を渡す。生徒の途中計算、誤り、修正も思考の一部として描く。
- 長いが機械的な計算は圧縮してよい。その場合も、出発式、実行した操作、得られた式を示し、「計算すると」だけで概念的な変形を隠さない。教師が計算結果を与えた場合は、生徒に代入、特殊ケース、数値例のいずれかで確かめさせる。
- 重要な式は、導出後に次元・単位、符号、既知の極限、簡単な特殊ケース、具体的数値のうち適切な方法で検算する。厳密式と近似式を区別し、近似が有効な条件を明記する。
- クライマックスでは、満たすべき条件を先に列挙し、単純な候補を式へ代入して違反を確認し、修正された候補へ絞る過程を省略しない。

## Markdown書式

- 話者名は `**やる夫**：` / `**やらない夫**：` を独立行に置く。注記は `**やる夫**（老いて）：` の形にする。
- 発言を外枠の `「` `」` で囲まず、継続行を全角空白で字下げしない。意味を持つ引用語のカギカッコは残す。
- 複数文の発言は原則として文末の `。` ごとに改行し、発言ブロックの最後の自然文も終止約物で閉じる。`？`・`！`・`……`・`──` などで終わる場合を除き、末尾に `。` を付ける（コード・数式・表・箇条書きで終わる発言は除く）。
- 金額は `100万円`、`1,000円`、`70ドル` のように半角算用数字で書く。割合は `40%`、`0.13%` のように半角算用数字と半角 `%` で書く。漢数字の金額、「パーセント」、全角数字・全角 `％` は使わない。
- ディスプレイ数式は `$$...$$`、台詞中の短い式は `$...$` を使う。
- 幕は `## 第N幕　タイトル`、節は `### N-M　小見出し ── 概念名`、区切りは `---` とする。
- 表は GFM 形式で書き、ヘッダ行の直後に `|---|---|` の区切り行を必ず置く。区切り行が無い、または列数がヘッダと合わないと表としてレンダリングされない。発言中に置く表も、直前に空行を1行入れる。
- 重要な到達点は太字、最終結果は `\boxed{}` を使う。強調 `**…**` は、閉じる `**` の直前が日本語の約物（`」`・`）`・`』`・`。`・`：` など）だと CommonMark の flanking 規則で効かず `**` がそのまま表示される。強調範囲に閉じ約物を含めない（`**快楽の踏み車**（…）` のように約物を外へ出す）か、閉じ `**` の直後へ半角スペースを1つ置く。
- 一つの台詞を長くても10行程度にし、教師の長広舌を分割する。ただし反応語だけの発言や、後続内容と結び付かない無内容な相槌で割らない。
