# 参照コーパスの選び方

作風や特殊な構成の実例が必要なときだけ読む。完成例を読むこと自体を作業目的にしない。

## 読み方

1. 下表から目的に近い1冊を選ぶ。
2. `rg '^#{2,3} ' docs/books/<file>.md` で見出しを確認する。
3. 必要な構造を示す1～3節だけを読む。
4. 全文を読むのは、ユーザーが明示した場合か、全体構成そのものを比較する場合に限る。

既存教材の続編・改訂では、その教材を第一の参照元とする。小さな改訂のために別の完成例を追加で読まない。
