# やる夫が日銀総裁になるようです ── 金融政策を再発明する

---

## 登場人物

**やる夫** ── ある日突然「明日から日銀総裁をやれ」と言われた男。経済の直観は悪くないが、すぐ「刷ればいい」と言って転ぶ。

**やらない夫** ── 金融政策の理論と歴史に通じた教師。答えを教えず、反例と史実で導く。

---

## 第1幕　「お金を刷ればインフレ」が壊れる日

---

**やる夫**：
やる夫、明日から日銀総裁だお。
仕事の内容は聞いてきたお。『物価の安定』だお。
……正直、楽勝だと思ってるお。

**やらない夫**：
ほう。根拠を言ってみろ。

**やる夫**：
物価ってのはお金とモノの交換比率だお。
世の中のお金が増えればお金の価値が下がって物価が上がる。
お金が減れば物価が下がる。
つまり、**デフレならお金を刷ればいいし、インフレなら回収すればいい**。
蛇口をひねるだけの仕事だお。シンプルイズベストだお。

**やらない夫**：
その考えには由緒正しい式がある。書けるか。

**やる夫**：
知ってるお。**貨幣数量説**だお

$$MV = PY$$

$M$ が貨幣量、$V$ が貨幣の流通速度、$P$ が物価、$Y$ が実質GDPだお。
$V$ と $Y$ が一定なら、$M$ を2倍にすれば $P$ も2倍。
だから蛇口をひねれば物価は動くお。

**やらない夫**：
よし。その式を具体例で動かしてみよう。
これから何度も使う例だから覚えておけ。**りんご島**だ。

人口100人の島。年にりんごを1万個生産し、それ以外に商品はない。
りんご1個の価格は100円。島に存在する貨幣は50万円。
この島の $V$ を計算しろ。

**やる夫**：
名目取引総額が $PY = 100 \times 10{,}000 = 100$万円。
貨幣は50万円だから

$$V = \frac{PY}{M} = \frac{1{,}000{,}000}{500{,}000} = 2$$

1枚のお札が年に平均2回、りんごの支払いに使われてるってことだお。

**やらない夫**：
では島の中央銀行が貨幣を50万円から100万円に倍増させた。
$V=2$、$Y=1$万個が変わらないなら、りんごの価格は？

**やる夫**：
$P = MV/Y = 2{,}000{,}000 / 10{,}000 = 200$円。
ちゃんと倍になるお。ほら、蛇口理論は正しいお。

**やらない夫**：
計算は正しい。では現実と照合しよう。
日本銀行は2013年、まさにお前の言う『蛇口をひねる』を史上最大規模でやった。
量的・質的金融緩和。マネタリーベース──日銀が直接供給するお金──は
2013年初の約135兆円から、10年で**約670兆円**、およそ5倍になった[^1]。
お前の理論なら物価はどうなっているはずだ。

**やる夫**：
5倍だお！　少なくとも数倍のインフレ、
年率に直しても2桁インフレが吹き荒れてるはずだお！

**やらない夫**：
実際の消費者物価上昇率は、2013年から2021年の平均で**年率0.5%前後**。
目標の2%にすら、ほぼ一度も安定的に届かなかった[^2]。

**やる夫**：
……？？
5倍刷って、物価はほぼ動かず……？
ちょっと待つお、式のどこが死んだのか調べるお。

$MV = PY$ で、$M$ が5倍、$P$ がほぼ不変、$Y$ もほぼ横ばいなら……

**$V$ が5分の1に潰れたんだお！！**
お金は増えたけど、増えた分がまったく使われずに、
どこかに積み上がってただけだお。

**やらない夫**：
その通り。増えたマネタリーベースの大半は、
銀行が日銀に持つ当座預金に積み上がったまま動かなかった。
りんご島で言えば、刷った50万円が島民の金庫に入ったきり、
一度もりんごの支払いに使われなかった状態だ。

**やる夫**：
うぐぐ……。
でも待つお、直すのは簡単じゃないかお？
$V$ が下がるなら、**下がった分だけ余計に刷ればいい**お。
$V$ が半分になったら $M$ を倍にする。蛇口を強くひねるだけだお。

**やらない夫**：
では判定してもらおう。
その処方箋が成り立つには、$V$ について何を仮定する必要がある？

**やる夫**：
えっと……$M$ を増やしたときに $V$ が**それ以上に下がらない**こと、
少なくとも $V$ の動きが**予測できる**ことだお。

**やらない夫**：
日本のデータはどうだった。$M$ を5倍にしたとき、$V$ は？

**やる夫**：
ほぼ正確に5分の1……。
あっ。これ、いたちごっこだお。
刷れば刷るほど、刷った分だけ $V$ が下がって吸収される。
**$V$ が $M$ と独立な定数だという仮定そのものが死んでる**んだお。

……ってことは、$MV=PY$ は嘘の式なのかお？

**やらない夫**：
式は嘘ではない。$V \equiv PY/M$ と**定義**すれば恒等式だ。
常に成り立つ。そして常に成り立つがゆえに、それだけでは何も予言しない。
予言をしていたのは『$V$ は安定な定数』という**追加の仮定**で、
死んだのはそっちだ。

**やる夫**：
恒等式に看板を掛けただけで、理論を持ってた気になってたのかお……。

じゃあ聞くお。**なんで $V$ は潰れたんだお？**
昔は安定してたんじゃないのかお。
教科書に載るくらいだから、どこかの時代では使えてたはずだお。

**やらない夫**：
良い問いだ。実際、19世紀から20世紀半ばまでは、
長期で見ればそれなりに安定して見えた。
そして『$V$ がいつ・なぜ潰れるのか』の答えは、
この教材の核心そのものだ。**第2幕でお前が自分で見つける**。
いま言えるのはヒントだけ──
**$V$ が潰れた時期、日本の短期金利は何%だった？**

**やる夫**：
ゼロ……ほぼゼロだお。ずっとゼロ。

**やらない夫**：
覚えておけ。もう一つ、先に問いを預けておく。
お前は明日から『物価の安定』が仕事だが、
日銀の目標は物価上昇率**2%** だ。
物価の『安定』なら**0%** が一番安定ではないのか？

**やる夫**：
…………言われてみれば変だお。
2%って毎年物価が上がるってことで、それは『安定』じゃなくて『一定の劣化』だお。
なんで0じゃなくて2なんだお？

**やらない夫**：
その答えは最終幕まで保留だ。自分の手で理論を組み上げれば、
2という数字が**恐怖から逆算された安全マージン**だと分かる。
保留リストに入れておけ。

**やる夫**：
保留その1：$V$ はなぜ・いつ潰れるのか。
保留その2：なぜ目標は0%でなく2%なのか。
……総裁就任前夜にして、蛇口理論が死んだお。
小手先で $M$ の量を調整する話じゃない。
**お金がなぜ使われたり使われなかったりするのか**、
そこから根本的にやり直す必要があるお。

**やらない夫**：
それが分かれば第1幕は合格だ。
次の幕では、お前が金庫に積んだままの50万円の気持ちを考えてもらう。

## 第2幕　金利ゼロの世界で、お金と国債が同じ紙になる

---

**やらない夫**：
まず、日銀が『お金を刷る』の実際の手順を確認する。
輪転機を回して札束をばら撒くわけではない。何をする？

**やる夫**：
知ってるお。**買いオペ**だお。
銀行が持ってる国債を日銀が買い上げて、代金を銀行の日銀当座預金に振り込む。
銀行の資産が『国債』から『お金』に入れ替わって、世の中のお金が増えるお。

**やらない夫**：
正確だ。つまり金融緩和とは、正体を言えば**資産の交換**だ。
民間から国債を吸い上げ、代わりにお金を渡す。
では思考実験だ。りんご島に戻る。

お前はりんご島の島民で、資産100万円を持っている。
内訳は現金50万円と、島の政府が発行する1年満期の国債50万円。
国債の金利は**年5%** だ。
そこへ中央銀行が来て『国債50万円を現金50万円と交換してくれ』と言う。
応じたら、お前の行動は変わるか？

**やる夫**：
変わるお。だって国債で持ってれば1年後に2万5千円の利息がついたのに、
現金は利息ゼロだお。
このまま現金で寝かせたら丸損だから、
何かに使うか、貸すか、別の資産を買うか──とにかく**動かす**お。

**やらない夫**：
そうだ。お前が現金を動かそうとする。島中の銀行や島民も同じことをする。
みんなが現金を手放して貸そうとするから貸出金利が下がり、
借金が安くなった誰かが果樹園に投資する。緩和が**効いた**状態だ。
では次。同じ交換を、**国債の金利が0%の世界**でやる。
現金50万円と、金利0%の国債50万円の交換。お前の行動は変わるか？

**やる夫**：
金利0%の国債……つまり1年後に50万円がそのまま返ってくるだけの紙だお。
現金も、1年置いておけば50万円のままの紙だお。

……ん？　同じだお。
どっちも『政府が価値を保証する、利回りゼロの紙切れ』だお。
白い紙を白い紙と交換されただけで、
やる夫のポートフォリオは**実質的に何も変わってない**お。
動かす理由が1ミリもないお。

**やらない夫**：
それが第1幕の保留その1の答えの正体だ。自分で言語化してみろ。
なぜ $V$ は潰れた？

**やる夫**：
！！
金利がゼロまで下がると、**お金と短期国債が完全に同じもの**になる。
その状態で買いオペをやっても、同じ紙を同じ紙と交換してるだけだから、
誰の行動も変わらない。
増えたお金は使われる理由がないから金庫（当座預金）に積み上がる。
つまり──**$M$ を増やした分がそっくり $V$ の低下に吸収される**。

第1幕でやる夫が見た『$M$ 5倍、$V$ 5分の1』は、
偶然じゃなくて**必然**だったんだお！

**やらない夫**：
完璧な言語化だ。これが**流動性の罠**。
1930年代にケインズが指摘し、多くの経済学者が『理論上の珍品』として
棚にしまっていた現象を、1990年代の日本が世界で初めて実演した[^3]。

**やる夫**：
ちょっと待つお。今の話、本当に金利ゼロだけが特別かお？
金利5%のときだって、買いオペで金利が4.9%に下がるだけなら
大した違いはないお。『効き目が弱い』と『効かない』の違いは
程度問題じゃないのかお？

**やらない夫**：
良い疑いだ。では対照実験で確かめろ。
金利5%の世界で、お前は現金をどれだけ財布に入れておく？

**やる夫**：
最小限だお。現金で持つと年5%損するから、
当面の買い物に必要な分だけ現金にして、残りは全部国債にするお。

**やらない夫**：
金利が5%から4%、3%、2%と下がっていくと？

**やる夫**：
現金で持つペナルティがどんどん軽くなるから、
だんだん現金を多めに持っても平気になる……。
で、**0%に達した瞬間、ペナルティが完全に消える**。
現金を100万円持とうが1億円持とうが、1円も損しない。

あっ、分かったお。5%→4%は『ペナルティの軽減』で、程度問題だお。
でもゼロは違う。**現金の保有量に対する需要が無限に伸びきる点**だお。
いくらお金を注ぎ込んでも、全部『とりあえず持っとくか』で吸収される。
例えるなら、底が抜けたバケツだお。
量的な変化じゃなくて、**相転移**だお。

**やらない夫**：
そうだ。だから金利がプラスの領域では『蛇口』にも一定の意味があったし、
$V$ もそれなりに安定して見えた。
教科書の貨幣数量説が『どこかの時代では使えていた』のはそのためだ。
だが金利ゼロの床──**ゼロ金利制約（ZLB: Zero Lower Bound）**──に
着地した瞬間、蛇口は空回りを始める。

**やる夫**：
なんで金利はゼロで止まるんだお？
マイナス5%とかにできれば、現金保有に思いっきりペナルティを
付けられるじゃないかお。

**やらない夫**：
銀行預金の金利をマイナス5%にしたら、お前はどうする。

**やる夫**：
即、全額下ろして現金でタンスに入れるお。現金の金利は常に0%……

**あー、そういうことかお**。
現金という『金利0%の逃げ場』が存在する限り、
金利は0%より大きくは下げられない。
紙幣の存在そのものが床を作ってるんだお。

**やらない夫**：
厳密には、現金の保管や輸送にはコストがかかるから、
床は0ちょうどではなく若干のマイナスまで掘れる。
実際、日銀は2016年に−0.1%[^4]、欧州中央銀行は−0.5%まで下げた[^5]。
だが『タンス預金への大脱走』が始まる限界までで、深くは掘れない。
床の存在は変わらない。

**やる夫**：
整理するお。

1. 緩和の正体は『お金と国債の交換』。
2. 金利がプラスなら、その交換は人々の行動を変える（効く）。
3. 金利がゼロだと、お金と国債は同じ紙になり、交換は無意味（効かない）。
4. そして金利は、現金の存在ゆえにゼロ近辺より下に行けない。

……つまり日銀の武器は金利で、その武器には**弾数の下限**があるんだお。
じゃあ聞くお。**床に着いてしまったら、中央銀行は完全に無力なのかお？**
現に日本は四半世紀近く床の上にいるお。総裁のやる夫は何をすればいいんだお。

**やらない夫**：
それが本教材最大の問いだ。**保留その3**として預かる。
答えは第5幕、お前が政策ルールを自力で導出したとき、
その式の中から自然に出てくる。
いま言えるのは方向だけ──
**今日の金利が動かせないなら、動かせるものは『今日』の外にある**。

**やる夫**：
今日の外……？　禅問答かお。

**やらない夫**：
第3幕でその禅問答を解く。
鍵は、お前がりんご1個買うときにすら無意識にやっている計算だ。

## 第3幕　今日の物価は明日の政策が決める

---

**やらない夫**：
りんご島に戻る。お前は島民で、手元に10万円ある。
今日、島の中央銀行が発表した。
『来年から3年間、貨幣を毎年2倍に増やし、島の景気を全力で支える』。
金利は今日時点でプラスとし、この宣言は完全に信用されるものとする。
りんごの価格は**今日**、どうなる？

**やる夫**：
来年からの話だお？　今日は貨幣量が1円も増えてない。
今日の $M$ が不変なら、今日の $P$ も不変……

……いや、待つお。
来年から貨幣が倍々に増えるって**確実に分かってる**なら、
来年のりんご価格は大きく上がるお。
ってことは、今日100円で買えるりんごは来年150円とか200円になる。
だったらやる夫は**今日のうちに買っておく**お。
……で、島民全員が同じことを考えるから、今日の時点で買いが殺到して、
**今日のりんご価格が上がる**お！　貨幣が1円も増える前に！

**やらない夫**：
そうだ。逆も確かめろ。
中央銀行が『来年から金融を思い切り引き締め、物価を下げる』と宣言し、
信用されたら？

**やる夫**：
来年安く買えると分かってるものを今日買うバカはいないお。
みんな買い控えるから、**今日の物価が下がる**。

……これ、すごく気持ち悪い結論だお。
今日の物価を動かしたのは、今日の貨幣量でも今日の金利でもない。
**将来の政策に関する『予想』** だお。
物価っていう現在の変数が、未来から逆流して決まってるお。

**やらない夫**：
気持ち悪さの正体を突き詰めろ。
物価とは何と何の交換比率だ。

**やる夫**：
お金とモノ……あっ。
物価が上がるってことは**お金の価値が下がる**ってことだお。
そしてお金は資産だお。1年後も持ち越せる。

資産の今日の価値は、**将来どうなるかの予想**で決まる。
株が『将来の利益の予想』で今日値付けされるのと同じで、
お金の価値＝物価の逆数も、
『将来お金がどれだけ刷られ、どれだけ稀少であり続けるか』の予想で
今日決まる。
物価は静的な需給の一致点じゃなくて、**資産価格**なんだお。

**やらない夫**：
鋭い。それが20世紀後半に金融政策論を作り替えた視点の転換だ。
中央銀行が本当に動かしているのは、今日の貨幣量という『点』ではない。
**将来にわたる政策の経路、その予想の全体**だ。
第2幕の禅問答を思い出せ。『今日の金利が動かせないなら──』

**やる夫**：
『──動かせるものは今日の外にある』。

！！　見えてきたお。
今日の金利がゼロに張り付いて動かせなくても、
**『明日以降の金利をどうするつもりか』という予想**は動かせる。
保留その3の答えはこの方向にあるんだお！？

**やらない夫**：
方向は正しい。だが今のお前には、その直観を政策に翻訳する道具がない。
試しに聞くが、『将来の緩和を予想させる』と言ったな。
具体的に、予想の**何が**、経済の**どこに**、**どう**効く？
りんご島の言葉で、因果の鎖を最後まで言えるか。

**やる夫**：
えっと……将来緩和するって予想させると、将来の物価が上がる予想になって、
だから今日買った方が得で……いや待つお、企業の投資はどうなるんだお？
金利の予想と物価の予想、どっちがどっちに効いて……
為替も動くお？　あれ、賃金は……

……ダメだお。矢印が絡まって、鎖の途中が全部ふわふわしてるお。

**やらない夫**：
それが正常だ。直観は掴んだが、直観だけでは政策は打てない。
誤差1%が数百万人の雇用を左右する仕事だからな。
必要なのは、絡まった矢印を1本ずつ固定する数学の道具だ。

第4幕では、それを4つ組み立てる。

1. **実質金利**──『金利』の中に2匹いることを見抜く道具。
2. **自然利子率**──金利の高い・低いを測る、見えない基準点。
3. **IS関係**──金利が需要を動かす鎖。
4. **フィリップス曲線**──需要が物価を動かす鎖。

この4つが揃うと、金融政策の全体が
『中央銀行→金利→需要→物価→予想→（最初に戻る）』という
1つの閉じたループとして書ける。
そしてそのループの上で、第5幕、お前は総裁としての行動原理を
**ほぼ一意に**導出することになる。

**やる夫**：
ほぼ一意に、かお。楽しみにしとくお。
保留リストを更新するお。

保留その1：~~$V$ はなぜ潰れるのか~~ → **解決**（ゼロ金利でお金と国債が同じ紙になるから）。
保留その2：なぜ目標は0%でなく2%か → 未解決、最終幕。
保留その3：金利の床に着いたら何ができるか → 方向だけ判明（予想に働きかける）、第5幕で具体化。

道具を作りに行くお。

## 第4幕　道具の構築 ── ループを1本ずつ固定する

### 4-1　金利の中に2匹いる ── 実質金利とフィッシャー方程式

**やらない夫**：
最初の道具だ。りんご島で、お前が島の銀行の融資担当だとする。
島民Aが『10万円貸してくれ。1年後に返す』と来た。
島のりんご価格は100円で安定しているとしよう。
金利何%なら貸す？

**やる夫**：
そうだお……ただ寝かせるよりは増えてほしいから、
仮に3%とするお。1年後に10万3千円。りんご換算で1,030個分だお。

**やらない夫**：
では条件を1つ変える。島は年10%のインフレで、
1年後のりんご価格は110円になると**確実に分かっている**。
さっきと同じ金利3%で貸すか？

**やる夫**：
1年後に返ってくる10万3千円で買えるりんごは
$103{,}000 / 110 = 936$個……

**減ってるお！！**
今日10万円は りんご1,000個分だから、
1,000個貸して936個返ってくる契約だお。
金利プラス3%のつもりが、りんごで測ると**マイナス6.4%** だお。
貸すどころか、金をドブに捨ててるお。

**やらない夫**：
つまり『金利3%』という数字だけでは、
その貸出が得か損かすら判定できない。
いま自分でやった計算を、一般式にしろ。
名目金利を $i$、インフレ率を $\pi$ とする。

**やる夫**：
円で測った増え方が $1+i$ 倍で、りんご1個の値段が $1+\pi$ 倍になるから、
**りんごで測った増え方**は

$$1 + r = \frac{1+i}{1+\pi}$$

$i$ や $\pi$ が小さければ、近似でこう書けるお

$$\boxed{\, r \approx i - \pi \,}$$

**やらない夫**：
それが**フィッシャー方程式**だ。
$i$ が**名目金利**──お札の枚数で測った金利。
$r$ が**実質金利**──モノの量で測った金利。
『金利』という言葉の中には常にこの2匹がいて、
人々の実体的な行動──借りるか、投資するか、消費を我慢するか──を
決めるのはどちらだ？

**やる夫**：
$r$ の方だお。さっきのやる夫は、りんご換算でマイナスだから貸すのをやめた。
人間が最終的に欲しいのはお札じゃなくてモノだから、
行動を支配するのは実質金利だお。

**やらない夫**：
一つ注意を足す。貸す時点では将来のインフレ率はまだ確定していない。
だから行動を決めるのは実際のインフレ率ではなく**予想インフレ率** $\pi^e$ だ

$$r = i - \pi^e$$

第3幕で『予想が物価を動かす』と言ったな。
予想が実体経済に入り込む正確な入口が、この $\pi^e$ だ。
では、この道具で日本を診断してもらおう。
1998年頃の日本。名目金利はほぼ0%、物価は年1%程度の**デフレ**。
巷では『史上最低金利、空前の金融緩和』と言われていた。
総裁として正しい診断を下せ。

**やる夫**：
$i = 0$、$\pi^e = -1\%$ として

$$r = 0 - (-1) = +1\%$$

実質金利は**プラス1%**……。
あれ、これって高いのかお？　低いのかお？
プラス1%が『緩和的』かどうか、比べる相手がいないと判定できないお。

**やらない夫**：
素晴らしい詰まり方だ。それがまさに次の道具への入口だ。
ただ、その前に片方だけ確定させておく。
デフレが**加速して**、$\pi^e$ が $-2\%$、$-3\%$ と落ちていくと、
$i=0$ のままなら $r$ は？

**やる夫**：
$r = 0-(-2) = +2\%$、$0-(-3)=+3\%$……**勝手に上がっていく**お。

うわ、これ最悪の構造だお。
名目金利は床（ZLB）に張り付いて下げられないのに、
デフレ予想が深まるほど実質金利は自動的に**引き締まっていく**。
景気が悪い→デフレが進む→実質金利が上がる→もっと景気が悪くなる→……
中央銀行がハンドルから手を離してるのに、車が勝手に逆走加速するお。

**やらない夫**：
それが**デフレスパイラル**の金利面の正体であり、
1933年にアーヴィング・フィッシャー──さっきの式の主──が
大恐慌の只中で『負債デフレ理論』として書いた悪夢だ。
物価が下がると、借金の**実質**負担が膨らむ。
1930年代の米国では物価が3割近く下がり、
『借金を返すために資産を投げ売りし、投げ売りが物価をさらに下げ、
残った借金がさらに重くなる』という循環で銀行と企業が連鎖倒産した。
フィッシャー本人も1929年の株価暴落で全財産を失っている[^6]。

**やる夫**：
デフレって『物価が下がってお得』じゃなくて、
借金してる企業と家計の首が実質的に締まっていく現象なんだお……。
保留その2の『なぜ0%でなく2%か』、
なんとなくこのあたりに絡んでる気配がするお。

**やらない夫**：
気配は正しいが、最終幕まで保留のままだ。
次は、お前がさっき詰まった問い──
『実質金利プラス1%は高いのか低いのか』──に答える道具を作る。

### 4-2　見えない基準点 ── 自然利子率とヴィクセルの暴走実験

**やらない夫**：
りんご島から一旦、貨幣を取り上げる。
物々交換……は不便だから、**りんごの現物貸借**だけがある島を考えろ。
今年のりんごを貸し、来年のりんごで返してもらう。
『りんご100個借りて、来年103個返す』なら、りんご建て金利3%だ。
この金利は何と何のバランスで決まる？

**やる夫**：
貸したい人と借りたい人の需給だお。
貸す側は『今年食うのを我慢する』人たち。
借りる側は……果樹園に投資したい人たちだお。
苗木を植えれば来年もっとりんごが採れるから、
増えた分から利息を払っても元が取れる。

だから、**我慢の対価**と**投資の実り**が釣り合う点で金利が決まるお。

**やらない夫**：
その、貨幣が一切登場しない世界で決まる実質金利を
**自然利子率** $r^*$ と呼ぶ。
経済の実力──技術（投資がどれだけ実るか）と、
人々のせっかち度（どれだけ我慢を嫌うか）と、人口──だけで決まる。
では貨幣を島に戻す。今度は銀行が貨幣を貸す。
中央銀行は銀行経由で、**貨幣建ての実質金利 $r$** を
$r^*$ からズラすことができる。
ここで実験だ。島の $r^*$ が3%のとき、
中央銀行が実質金利を**1%に固定し続けた**ら何が起きる？

**やる夫**：
投資の実りは3%あるのに、1%で借りられる……
借りたもん勝ちだお！　島民がこぞって借りて果樹園を作りまくる。
需要が実力以上に膨らむお。

**やらない夫**：
膨らんだ需要は物価をどうする。

**やる夫**：
苗木も労働者も奪い合いになるから、物価と賃金が上がる。
インフレだお。……で、それだけかお？
インフレが起きたら需要の過熱は収まるんじゃないかお。

**やらない夫**：
収まるかどうか、フィッシャー方程式で判定してもらおう。
中央銀行が固定しているのが**名目**金利だったとしよう。
こちらの方が現実の操作に近い。
$r^* = 3\%$、当初インフレ予想0%で、名目金利を1%に固定する。
初期の実質金利は？

**やる夫**：
$r = i - \pi^e = 1 - 0 = 1\%$。$r^* = 3\%$ より低いから過熱スタートだお。

**やらない夫**：
過熱でインフレが起き、島民のインフレ予想が2%に上がった。
名目金利は1%に固定したままだ。実質金利は？

**やる夫**：
$r = 1 - 2 = -1\%$……**さっきより下がってるお！？**
緩和度合いが**勝手に深まってる**お。
てことは過熱がさらに激しくなって、インフレ予想が4%に上がって、
$r = 1 - 4 = -3\%$、もっと過熱して、$\pi^e = 6\%$ で $r = -5\%$……

**止まらないお。加速する一方だお！！**

**やらない夫**：
逆向きも確認しろ。名目金利を $r^*$ より**高く**固定したら。

**やる夫**：
実質金利が高すぎる→需要不足→デフレ→ $\pi^e$ 低下→
実質金利 $r = i - \pi^e$ が**さらに上がる**→もっとデフレ→……
今度は下向きに止まらないお。

……分かったお、この系の構造。
名目金利の固定って、**棒を手のひらに立てるのと同じ**だお。
$r = r^*$ の一点だけが釣り合いで、
1ミリでもズレたら、ズレを拡大する向きに力が働く。**不安定平衡**だお。

**やらない夫**：
その暴走メカニズムを1898年に『利子と物価』で定式化したのが
スウェーデンのクヌート・ヴィクセルだ。**累積過程**と呼ばれる[^7]。
そして20世紀は、この理論の警告を無視して実演までやった。
第二次大戦中の1942年、米国FRBは戦費調達を助けるため、
国債の金利に上限を設けて釘付けにする政策を財務省に約束した。
戦後もやめられずに続け、何が起きたと思う。

**やる夫**：
金利固定＝手のひらの棒だから……インフレが暴走したのかお。

**やらない夫**：
1947〜48年に米国のインフレ率は一時14%を超えた。
朝鮮戦争勃発で再びインフレが吹き上がるに及んで、
FRBはついに財務省と衝突し、1951年の**アコード（合意）** で
釘付けから解放された[^8]。
この事件は『中央銀行の独立性』が教科書の題目になった原点の一つだ。
政府は常に低金利で借金したがる。それに付き合って金利を釘付けにすれば、
ヴィクセルの棒は倒れる。

**やる夫**：
歴史が反例をやってくれてるんだお……。
でも待つお、じゃあ話は簡単じゃないかお？
**$r = r^*$ になるように金利を合わせ続ければいい**。
棒が倒れないよう、支点を真下に保つだけだお。
$r^*$ の値さえ分かれば……

……あれ。$r^*$ って、どうやって測るんだお？
『貨幣のない仮想の島で決まる金利』なんて、観測できないお。

**やらない夫**：
できない。$r^*$ は推計するしかなく、推計値は手法によって
1%ポイント単位で平気でブレる。
しかも技術や人口動態でゆっくり動く。
これは第5幕で政策ルールを選ぶとき、
**『観測できない変数に頼りすぎるルールは危険』** という
要請の形で戻ってくる。覚えておけ。

もう一つ、$r^*$ について不吉な事実を予告しておく。
先進国の $r^*$ の推計値は、過去40年、ほぼ一貫して**下がり続けている**。
これが何を意味するかは最終幕だ。

**やる夫**：
4-1の詰まりに答えが出たお。
『実質金利プラス1%は高いのか低いのか』──**$r^*$ と比べろ**、が答えだお。
もし日本の $r^*$ が0%近くやマイナスまで落ちてたなら、
実質金利プラス1%は『史上空前の緩和』どころか**引き締め**だったんだお。
巷の『低金利なのに効かない』は、そもそも低金利じゃなかった……

**やらない夫**：
診断の道具が2つ揃った。
次は、実質金利が需要を動かす鎖──ISの鎖を固定する。

### 4-3　我慢の値段 ── IS関係

**やらない夫**：
3本目の鎖。実質金利が変わると、今日の需要はなぜ・どれだけ動くか。
りんご島のお前個人の問題として解く。
お前は今年1,000個、来年も1,000個のりんご収入がある。
りんご建て実質金利 $r$ で貸し借りできる。
$r = 0\%$ のとき、お前は今年何個食べる？

**やる夫**：
収入が平らで、貸しても増えないなら、そのまま今年1,000個・来年1,000個だお。
空腹の年と満腹の年を作るより、平らに食べる方が満足度が高いお。

**やらない夫**：
では $r = 20\%$ に跳ね上がった。今年の食べ方は変わるか？

**やる夫**：
今年100個我慢すれば来年120個になる……。
20個のボーナスは大きいお。今年を950個に減らして50個貸し、
来年は1,060個食べる、みたいな配分にシフトするお。

つまり実質金利って、**『今日の消費』の値段**なんだお。
今日1個食べることは、来年 $1+r$ 個食べる権利を捨てること。
$r$ が高いほど今日の消費は高くつくから、今日の需要が減る。

**やらない夫**：
企業側も同じ構造だ。果樹園への投資は
『実りの率が $r$ を上回るなら実行』だから、$r$ が上がるほど
実行される投資案件は減る。
家計の異時点間の選択と企業の投資判断、両方を束ねると、
島全体でこういう関係が立つ

$$x = -\sigma\,(r - r^*)$$

$x$ は**需要ギャップ**──経済の実力（潜在GDP）に対して
需要が何%過剰・不足かだ。$\sigma > 0$ は反応の強さ。
この式を**IS関係**と呼ぶ。言葉で読め。

**やる夫**：
実質金利 $r$ が自然利子率 $r^*$ より高ければ需要は実力を下回り（$x<0$）、
低ければ実力を上回る（$x>0$）。ぴったりなら需要は実力通り（$x=0$）。

4-2の手のひらの棒が、そのまま式になってるお。
$r - r^*$ っていう**差**だけが効くのがポイントだお。
金利の絶対水準に『緩和的』も『引き締め的』もない。

**やらない夫**：
一つだけ精密化する。お前がさっき食べ方を決めたとき、
本当は今年の $r$ だけでなく、**来年以降の $r$ の予想**にも依存するはずだ。
5年間ずっと高金利が続くと予想すれば、今日の我慢はさらに深くなる。
厳密なIS関係は『現在から将来にわたる実質金利経路の全体』で
今日の需要が決まる形になる。
第3幕の『物価は未来から逆流して決まる』が、
需要についても成り立つわけだ。この形は第5幕で効いてくる。

### 4-4　値札はなぜサボるのか ── 粘着価格とフィリップス曲線

**やる夫**：
ちょっと待つお。ここまで作ってきて、根本的な矛盾に気づいたお。

需要ギャップ $x$ がプラスやマイナスになる、って前提で話してるけど、
**そもそもそんなこと起きるのかお？**
需要が実力を下回るなら、売れ残った店は値下げするお。
価格が下がれば買い手が戻る。
市場価格ってそのためにあるんだお。
価格が瞬時に調整されるなら、需要はいつも実力ぴったり、
$x$ は恒等的にゼロ。ISの式なんて出番がないお。

**やらない夫**：
これ以上ない良い異議だ。その主張には名前まである。**古典的二分法**──
価格が完全に伸縮的なら、貨幣や金利は実体経済（生産・雇用）に
一切影響せず、物価だけを動かす。
ではその主張を、判定してもらおう。
もし古典的二分法が正しいなら、
中央銀行がどれだけ猛烈な引き締めをやっても、
失業は増えないはずだな？

**やる夫**：
物価がスッと下がって終わり、実体は無傷のはずだお。

**やらない夫**：
1979年、米国のインフレ率は13%を超えていた。
FRB議長に就任したポール・ボルカーは、
インフレ退治のため史上最強級の引き締めを敢行し、
政策金利は一時**20%** 近くに達した。
結果、インフレ率は1983年までに3%台へ急落した。
同時に何が起きたか──失業率は1982年に**10.8%**、
大恐慌以来の水準まで跳ね上がり、深刻な景気後退が起きた[^9]。

**やる夫**：
物価『だけ』は動かなかった……。
引き締めの衝撃を、まず**生産と雇用**がまともに食らって、
物価はその後からゆっくり降りてきたんだお。
つまり価格は瞬時に調整**されてない**。事実として。

でも、なんでだお？　値下げなんて値札を書き換えるだけだお。
売れ残りを抱えるより値札を書き換える方が絶対マシだお。

**やらない夫**：
では、お前がりんご島で唯一のりんごジュース屋だとしよう。
1杯300円。原価や客足が日々小刻みに揺れる中で、
お前は本当に**毎日**値札を書き換えるか？

**やる夫**：
毎日はやらないお。
メニューの刷り直しは面倒だし、
300円が298円になったところで客は増えないし、
逆に頻繁に値上げ値下げすると常連が
『この店、足元を見てくる』って不信感を持つお。
原価が多少動いても、**よほどのことがない限り据え置く**お。

**やらない夫**：
その『面倒』『客の不信』が**メニューコスト**だ。
1回1回は小さなコストだが、
全店が『よほどのことがない限り据え置く』を選ぶ結果、
経済全体の物価は需要の変化に対して**ゆっくりしか**動かなくなる。
個々の店の合理的なサボりが、集計すると価格の**粘着性**になる。
では、お前がジュースの価格を改定する数少ない機会に、
何を見て新価格を決める？

**やる夫**：
今の原価と客足……だけじゃないお。
次の改定までしばらく据え置くつもりなんだから、
**据え置き期間中の原価や物価がどうなりそうか**を織り込むお。
周りの物価が年2%上がっていきそうなら、
最初からちょっと高めに付けておくお。

**やらない夫**：
その行動を全店について集計すると、インフレ率はこう書ける

$$\pi = \pi^e + \kappa\, x$$

**フィリップス曲線**だ。読め。

**やる夫**：
今日のインフレ率は、
（1）みんなが予想するインフレ率 $\pi^e$ を土台に、
（2）需要の過熱・不足 $x$ の分だけ上下する。
$\kappa$ は価格が粘着的なほど小さい……
値札がサボりがちなほど、需要が動いても物価はちょっとしか動かない。

あ、これでボルカーの現象が説明できるお。
引き締め→需要激減（$x$ が大きくマイナス）→
でも $\kappa$ が小さいから物価はすぐには落ちない→
落ちない間、実体経済が $x<0$ の痛みを負担し続ける→
時間をかけて $\pi^e$ 自体が下がってやっと物価が落ち着く。
**価格がサボった分の調整を、生産と雇用が肩代わりする**んだお。

**やらない夫**：
そして、その肩代わりこそが**金融政策が実体経済に効く理由**だ。
二分法の世界なら中央銀行は物価係にすぎないが、
価格が粘着的な現実では、金利は雇用と生産を実際に動かす。
総裁の仕事が重いのは $\kappa$ が小さいからだ。
――さて、これで部品が4つ揃った。組み立てろ。

**やる夫**：
並べるお。

$$r = i - \pi^e \quad(\text{フィッシャー})$$
$$x = -\sigma(r - r^*) \quad(\text{IS})$$
$$\pi = \pi^e + \kappa x \quad(\text{フィリップス曲線})$$

鎖をつなぐと：
**日銀が名目金利 $i$ を動かす → （予想 $\pi^e$ を介して）実質金利 $r$ が変わる
→ $r$ と $r^*$ の差が需要ギャップ $x$ を動かす → $x$ がインフレ $\pi$ を動かす
→ 実現した $\pi$ が予想 $\pi^e$ に還流する → 最初に戻る**。

第3幕で絡まってた矢印が、1本のループになったお。

**やらない夫**：
そのループが、現代のニューケインジアン・モデルの骨格だ。
実物の摩擦や金融システムを削ぎ落とした最小骨格だが、
政策の論理を考えるにはこれで足りる。
残る道具はあと1つ──このループの上で日銀は
**どういう規則で $i$ を動かすべきか**。
その答えの候補と、候補を殺す判定法を作る。

### 4-5　1対1では足りない ── テイラー原理

**やらない夫**：
本題の前に、素朴な問いから入る。
インフレ率が目標より1%高くなった。名目金利をどうする？

**やる夫**：
上げるお。過熱を冷ますんだから当然だお。
で、インフレが1%上振れたんだから、**金利も1%上げる**。
1対1で対応する。素直で美しいお。

**やらない夫**：
では判定してもらおう。りんご島で数値実験だ。
$r^* = 1\%$、インフレ目標2%、$\sigma = 1$、$\kappa = 0.5$ とする。
定常状態の名目金利は？

**やる夫**：
$i = r^* + \pi = 1 + 2 = 3\%$ だお。
このとき $r = 3 - 2 = 1\% = r^*$ で、$x = 0$、インフレは2%のまま。釣り合ってるお。

**やらない夫**：
そこに撹乱が入る。何かの拍子に人々のインフレ予想が
2%から3%に上がってしまった。
お前のルール通り、金利を1%上げて4%にする。実質金利は？

**やる夫**：
$r = i - \pi^e = 4 - 3 = 1\%$。

……あれ？　$r^*$ と同じ、つまり**元のまま**だお。
需要ギャップは $x = -\sigma(r - r^*) = 0$。過熱を全然冷やせてないお。

**やらない夫**：
フィリップス曲線で実際のインフレは？

**やる夫**：
$\pi = \pi^e + \kappa x = 3 + 0 = 3\%$。
予想通りの3%が**実現**しちゃったお。
するとみんな『ほら、やっぱりインフレは3%だ』って自信を深めて……
次は予想が4%に上がっても同じことが起きて、5%でも、6%でも……

**！！　まずいお。これ、根無し草だお。**
やる夫の1対1ルールだと、実質金利が絶対に動かないから、
インフレ予想がどこへ漂流しても、それをそのまま追認するだけだお。
物価をつなぎ留める錨がどこにもないお。

**やらない夫**：
原因を自分の言葉で言え。1対1の何がダメだった。

**やる夫**：
効くのは実質金利なのに、名目で1対1に追いかけると
$r = i - \pi^e$ の引き算で相殺されてゼロ回答になるからだお。
これ……**4-2のヴィクセルの棒と同じ病気だお！**
金利釘付けは論外だと思ってたけど、
1対1ルールは『予想と一緒に動く釘付け』でしかなくて、
実質金利が固定される点では同罪だったんだお。

**やらない夫**：
良い転移だ。では処方箋を書け。

**やる夫**：
インフレが1%上振れたら、名目金利は**1%より大きく**上げる。
例えば1.5%上げれば、実質金利が0.5%上がって、需要を実際に冷やせる。
式にするお。インフレの上振れ $(\pi - \pi^{目標})$ に対して

$$i = r^* + \pi^{目標} + \phi\,(\pi - \pi^{目標}), \qquad \phi > 1$$

係数 $\phi$ を**1より大きく**する。これが錨の条件だお。

**やらない夫**：
その $\phi > 1$ が**テイラー原理**だ。
1993年にジョン・テイラーが、実際のFRBの行動をよく再現する
経験則として提示したルール──**テイラー・ルール**──の心臓部にあたる[^10]。
テイラーの元の式は生産面も見る

$$i = r^* + \pi^{目標} + 1.5\,(\pi - \pi^{目標}) + 0.5\, x$$

では新しい判定装置の試運転だ。合格例と不合格例、両方やれ。
まず $\phi = 1.5$。さっきと同じく予想が3%に上振れた直後の1周を計算しろ。
簡単化のため $x$ への反応は無視してよい。

**やる夫**：
金利は $i = 1 + 2 + 1.5 \times (3-2) = 4.5\%$。
実質金利 $r = 4.5 - 3 = 1.5\%$ で、$r^*=1\%$ を **0.5%上回る**。
需要は $x = -1 \times 0.5 = -0.5\%$ と冷えて、
インフレは $\pi = 3 + 0.5 \times (-0.5) = 2.75\%$。

予想の3%より**低い**インフレが実現したお。
『あれ、思ったほど上がらないな』ってなって、予想は3%から下方修正される。
修正されたらまた同じ力学で押し下げられて……**2%に吸い寄せられていく**お。
合格だお。

**やらない夫**：
不合格例。$\phi = 0.8$ で同じ計算。

**やる夫**：
$i = 1 + 2 + 0.8 \times 1 = 3.8\%$。
$r = 3.8 - 3 = 0.8\%$ で、今度は $r^*$ を **0.2%下回る**お。
冷やすつもりが逆に燃料を足してるお！
$x = -1 \times (-0.2) = +0.2\%$、
$\pi = 3 + 0.5 \times 0.2 = 3.1\%$。
予想3%を**上回る**3.1%が実現→予想がさらに上方修正→さらに超過→……
**発散だお。** 不合格。

まとめるお。$\phi$ と1の大小で系の運命が真っ二つに割れる。

- $\phi > 1$：インフレの上振れに実質金利の**上昇**で応じる → 上振れは自己減衰 → **安定**
- $\phi \leq 1$：実質金利が動かないか逆走する → 上振れが自己増殖 → **漂流・発散**

棒を手のひらに立てるんじゃなくて、
**糸で天井から吊るす**構造に変えるのが $\phi > 1$ なんだお。
ズレたら押し戻される向きに力がかかる。

**やらない夫**：
道具は揃った。振り返れ。
お前はいま、フィッシャー方程式・自然利子率・IS関係・フィリップス曲線という
4つの部品と、テイラー原理という判定装置を持っている。
第5幕では、総裁やる夫の**政策運営ルールそのもの**を、
条件の列挙と候補の棄却によって導出する。
そこで保留その3──床に着いたら何ができるか──も回収する。

## 第5幕　総裁の行動原理を導出する（クライマックス）

---

**やらない夫**：
就任前の最終試験だ。
『日銀総裁やる夫は、どういう規則に従って政策を運営すべきか』。
解き方は理論構築の王道でいく。
まず、良い政策運営が**満たすべき条件**を番号付きで列挙しろ。
第1幕からの全教訓を使え。

**やる夫**：
列挙するお。

**(1) 物価の錨**：インフレ率（とその予想）が特定の目標値に
つなぎ留められること。予想の漂流や自己増殖を許さない（4-5の教訓）。

**(2) 実体の安定**：需要ギャップ $x$ の振れをできるだけ小さくすること。
価格が粘着的な世界では、物価調整の痛みを雇用と生産が肩代わりするから（4-4）。

**(3) 観測可能性**：ルールが参照する変数は、観測または合理的に推計できること。
観測不能な変数に精密に依存するルールは実行できない（4-2の $r^*$ の教訓）。

**(4) 頑健性**：$V$ のような『安定してるはず』の関係が崩れても
壊れないこと（第1幕の教訓）。

**(5) ZLB対応**：名目金利が床に着いた状態でも、無力にならないこと（第2幕）。

**(6) 信頼可能性**：ルールは公表でき、人々に信じてもらえる形であること。
効き目の入口が予想 $\pi^e$ である以上、信じられないルールは効かない（第3幕）。

**やらない夫**：
良いリストだ。では候補を単純な順に審査する。
**候補A：裁量**。ルールなど決めず、その時々で総裁が最善を判断する。
一見、これが一番柔軟で賢そうだが。

**やる夫**：
柔軟なのは認めるお。でも条件(6)で即死だお。
『その時々で判断します』は、人々から見れば
**将来の政策が予想不能**ってことだお。
インフレ予想のつなぎ留め（条件(1)）は、
『中央銀行はこう動くはずだ』という予想の共有があって初めて成立する。
しかも裁量には、もっといやらしい問題があるお。
……選挙前の政府が『ちょっとだけ緩和してくれ』と言ってきたとき、
ルールという盾がない総裁は断る根拠を失うお。
1942年のFRBが財務省に飲み込まれた構図（4-2）の再来だお。

**やらない夫**：
後者は**時間非整合性**の問題として1970年代末に定式化され、
『ルール対裁量』論争として一時代を築いた[^11]。
では**候補B：フリードマンのk%ルール**。
貨幣量 $M$ を毎年一定率 $k$%（例えば3%）で機械的に増やす。
マネタリズムの総帥ミルトン・フリードマンが提唱し、
『中央銀行の裁量こそ経済不安定の元凶。コンピュータに置き換えろ』
とまで言った由緒あるルールだ。条件(6)は満点──完全に予測可能だ[^12]。

**やる夫**：
予測可能性は満点でも、第1幕の亡霊が出るお。
k%ルールが物価を安定させるには $MV = PY$ の $V$ が安定してる必要がある。
条件(4)で審査：$V$ が崩れたら？
$M$ を律儀に3%増やし続けても、$V$ が勝手に動けば物価は漂流するお。
……これ、史実はあるのかお？　頭のいい人たちが試したはずだお。

**やらない夫**：
ある。1970年代の高インフレに懲りた各国は、
80年代前後にこぞって貨幣量目標を導入した。
そして金融自由化・技術革新で貨幣需要が激しく動き、
$V$ が予測不能になって、90年代までに主要国は軒並み放棄した。
カナダ銀行のブーイー総裁が残した台詞が墓碑銘として名高い──
**『我々が貨幣供給量を見捨てたのではない。貨幣供給量が我々を見捨てたのだ』**[^13]。

**やる夫**：
一流の実務家が実地で試して死んだ案なら、やる夫が拾い直す義理はないお。
候補B、条件(4)で棄却。

なら**候補C**だお。第4幕で作ったやつをそのまま使う。
**テイラー・ルール、ただし $\phi > 1$**

$$i = r^* + \pi^{目標} + \phi(\pi - \pi^{目標}) + \phi_x\, x, \qquad \phi > 1$$

条件(1)：テイラー原理で錨は効く。合格（4-5で検証済み）。
条件(2)：$x$ にも反応する。合格。
条件(6)：式1本で公表できる。合格。
条件(4)：$V$ が何をしようが式に $M$ が出てこないから無傷。合格。
……ほぼ満点だお？　これで決まりじゃないかお。

**やらない夫**：
条件(3)と(5)の審査がまだだが。

**やる夫**：
うぐ。(3) 観測可能性……式の中に **$r^*$ と $x$ がいる**お。
どっちも観測できない推計値で、4-2でやった通り $r^*$ の推計は
1%ポイント単位でブレる。
$r^*$ を1%読み間違えたルールは、常時1%ズレた金利を機械的に打ち続ける……。

それから(5)。ZLB。
インフレがマイナスに落ちて、式が $i = -2\%$ とか要求してきたら、
床があるから実行できない。
実行できないどころか、4-1の悪夢が始まるお。
$i$ は0で止まってるのにデフレで $\pi^e$ が下がり続けると、
実質金利が勝手に**上がって**いく。
ルールが『緩和しろ』と叫んでるのに、現実は自動引き締め。
条件(5)、そのままでは**不合格**だお。

**やらない夫**：
では、テイラー・ルールを土台に、(3)と(5)を埋める修理をしろ。
まず(5)から。第3幕の禅問答の答えを、道具を使って完成させる時間だ。
今日の $i$ が床に張り付いて動かせない。
4-3の最後に精密化しただろう──今日の需要は何で決まる？

**やる夫**：
今日だけじゃなく、**将来にわたる実質金利経路の全体**で決まる……。

！！　つながったお。
今日の $i$ は動かせなくても、**明日以降の $i$ の予想**は約束で動かせる。
『景気が回復して、本来なら利上げすべき状況になっても、
**なお当分ゼロ金利を続ける**』と約束するお。
そうすれば将来の実質金利の予想が下がって、
経路全体としての緩和度が深まって、**今日の需要が動く**。
しかも将来の緩和はインフレ予想 $\pi^e$ を引き上げるから、
フィッシャー方程式経由で**今日の実質金利そのもの**も下がる。
$i$ が1ミリも動かなくても、$r = i - \pi^e$ は下げられるんだお！

**やらない夫**：
それが**フォワードガイダンス**（時間軸政策）だ。
保留その3、正式に回収と認める。
付け加えると、これは机上の空論ではなく、
世界で最初に実戦投入したのは**日銀**だ。
1999年、ゼロ金利政策の導入に際して
『デフレ懸念の払拭が展望できるまで』継続すると表明した[^14]。
理論の側では、ちょうど同じ頃クルーグマンが日本の流動性の罠を分析し、
この政策の核心を鮮烈な一句に要約している──
中央銀行は **『無責任になることを、信じてもらえる形で約束せよ』**[^15]。

**やる夫**：
無責任に……？　一瞬ぎょっとするけど、意味は分かるお。
普通の中央銀行は『インフレの芽は早めに摘む責任感の塊』だと思われてる。
でもZLBでは、その責任感が仇になるお。
『どうせ景気が戻ったら即引き締めるんでしょ』と思われたら、
将来の緩和の約束を誰も信じない → $\pi^e$ が上がらない → 今日の緩和も効かない。
だから『景気が戻っても**しばらく緩和をやめない**』という
“普段なら無責任”な行動を、あえて確約する必要があるんだお。

……でも、それってすごく難しい約束だお。
いざ景気が回復したら、その時の日銀には引き締めたい誘惑が生まれる。
みんなそれを見越してるから、口先の約束じゃ信じてもらえない。
条件(6)の信頼可能性が、ZLBでは何倍も重くなるお。

**やらない夫**：
その難しさの認識ごと正解だ。だからこそ実務では、
約束を縛る仕掛け──物価目標との紐付け、実績連動の条件文、
長期国債の大量購入という『引き返しにくさ』の演出──が
積み重ねられてきた。2013年の日銀の政策パッケージは、
その仕掛けの総動員だったと言える。
では最後、条件(3)の修理。$r^*$ と $x$ の観測不能性はどうする。

**やる夫**：
完全な解決は無理だお。だから方針は2つだお。
第一に、**推計を誠実にやり続ける**こと。潜在GDPも $r^*$ も、
複数の手法で推計して幅ごと扱う。点じゃなくて帯で見る。
第二に、**ルールを機械実行しない**こと。
式は骨格として公表しつつ、推計誤差の分だけ判断の余地を残す。

あれ……でもこれ、候補Aの裁量に逆戻りしてないかお？

**やらない夫**：
良い自問だ。逆戻りしないための現代の答えが、
ルールと裁量の中間形態──**インフレ目標政策**（インフレーション・ターゲティング）だ。
構造はこうだ。

**目標は固く**：物価目標（例：2%）を数値で公表し、達成に説明責任を負う。
**手段は柔らかく**：金利経路はテイラー・ルール的な考え方を骨格に、
推計の不確実性を踏まえて判断する。
『制約された裁量』とも呼ばれる。
1990年にニュージーランドが世界で初めて法制化し、
カナダ、英国、スウェーデンと続き、いまや先進国の標準装備だ[^16]。
日銀も2013年1月に2%目標を正式採用した[^2]。

**やる夫**：
導出の全体をまとめるお。

条件(1)〜(6)を課すと、
候補A（純粋裁量）は(1)(6)で死に、
候補B（k%ルール）は(4)で死に、
候補C（素のテイラー・ルール）は(3)(5)で修理が必要になり、
修理を施すと──

$$\boxed{\;\text{数値インフレ目標} \;+\; \text{テイラー原理}(\phi>1)\text{に従う金利運営} \;+\; \text{ZLBではフォワードガイダンス}\;}$$

という形に**ほぼ一意に**追い込まれる。
現代の中央銀行が似たり寄ったりの枠組みなのは、
流行りだからじゃなくて、**条件を課すと残る道がこれしかない**からだお。

**やらない夫**：
それが第5幕の到達点だ。
残るは答え合わせ──理論を歴史と実データに照らす。
そして最後の保留、『なぜ2%か』を回収する。

## 第6幕　答え合わせ ── 日本という実験場

---

**やらない夫**：
理論は現実で検証されて初めて政策科学だ。
組み上げた枠組みで、歴史を4件、診断してもらう。
**症例1：ボルカーのディスインフレ（米国、1979–83）**。
インフレ13%超 → 政策金利20%近く → インフレ3%台、ただし失業率10.8%。
お前の道具で読め。

**やる夫**：
テイラー原理の強行発動だお。
名目金利をインフレ率よりはるかに大きく引き上げて、
実質金利を強烈なプラスにした（$\phi \gg 1$ 相当）。
IS関係で需要が凍りつき（$x$ が大幅マイナス）、
フィリップス曲線の $\kappa$ が小さいせいで、
物価が降りてくるまでの調整を雇用が肩代わりした──10.8%という形で。

教訓は2つだお。
テイラー原理は**効く**。予想の錨は力ずくでも打ち直せる。
でもその費用は巨大だから、**そもそも錨を流されるな**。
インフレ予想が漂流を始める前に対処するのが安上がりだお。

**やらない夫**：
**症例2：日本の1990年代末〜2012年**。
ゼロ金利（1999）、量的緩和（2001、世界初）、
いずれも解除と再開を繰り返し、デフレ基調は続いた[^14]。

**やる夫**：
第1・2幕そのものだお。
ZLBに世界で最初に着地して、
『お金と国債が同じ紙になる』世界で買いオペを積み増した。
$M$ は増えて $V$ が潰れた。
フォワードガイダンスの原型（時間軸）は発明したけど、
2000年と2006年に緩和を解除して、そのたびに
『どうせ早めに引き締める』という予想を裏書きしちゃったお。
クルーグマンの言う『無責任の約束』と逆に、
**責任感を証明し続けた**ことになる。信頼の使い方が逆だったんだお。

**やらない夫**：
後知恵で言えばそうなる。当時は $r^*$ がどこまで沈んでいるかの
認識も定まっていなかったことは、フェアネスのために言い添えておく。
**症例3：2013年、量的・質的金融緩和（黒田日銀）**。
2%目標の正式採用、マネタリーベース倍増、長期国債の大量購入、
『2年程度で2%』のコミットメント。
第1幕でお前はこれを『5倍刷って物価はほぼ動かず』と嘲笑した。
いま同じ診断か？

**やる夫**：
……診断を書き直すお。
第1幕のやる夫は『$M$ を増やす→$P$ が上がる』の蛇口理論で読んで、
失敗と断じた。
でも第5幕を経た目で見ると、あの政策の設計思想は蛇口じゃないお。
巨額の買いオペは、それ自体で物価を押すためというより、
**『2%までやめない』という約束の担保**──
引き返しにくさの演出、フォワードガイダンスの重しだったんだお。

実際、効果もゼロじゃなかった。デフレ（物価の持続的下落）は止まって、
円安と雇用改善も起きた。
ただ、インフレ予想を2%に**再固定する**ところまでは届かなかった。
15年デフレで固まった『物価は上がらないもの』という予想は、
想定よりはるかに粘着的だったお。
$\pi^e$ って、動かすのは難しくて、
一度固まった錨は**下向きにも**強力なんだお。

**やらない夫**：
その後の展開も併せて記録しておく。
2016年にマイナス金利と長短金利操作（YCC）へ枠組みを拡張。
転機は外から来た。2022年以降、資源価格と円安で輸入物価が急騰し、
世界的インフレの波及で日本のインフレ率も2%を大きく超えた。
賃金と物価がともに動き出したのを確認して、
日銀は2024年3月、マイナス金利とYCCを解除し、
17年ぶりの利上げで金利ある世界への出口に踏み出した[^14]。
**症例4はその世界インフレだ。2021–23年、米欧のインフレは
一時8〜10%に達した。**[^17]

**やる夫**：
症例1の教訓の再演だお。
パンデミック後の供給制約と大規模財政で $x$ が急拡大したのに、
各国中銀は当初『一時的』と読んで利上げが遅れた。
テイラー原理の観点では、$\pi$ の上振れに $i$ が追いつかない期間、
実質金利はむしろ**下がって**火に油だったお。
遅れて急激な連続利上げ（テイラー原理の回復）で錨を打ち直した。
ボルカー期ほどの失業急増なしに沈静化に向かったのは、
インフレ予想の漂流が本格化する前に引き締めたから──
つまり『錨は流され切る前なら安く打ち直せる』の実証だお。

**やらない夫**：
では最後の保留を回収する。
**保留その2：物価の安定が目標なら、なぜ0%でなく2%なのか。**
お前はもう、自力で答えられるはずだ。第2幕と4-1を接続しろ。

**やる夫**：
……つながったお。鍵は**ZLBからの距離**だお。

定常状態の名目金利は $i = r^* + \pi^{目標}$。
これが、不況が来たときに**利下げできる余地**の大きさだお。
仮に $r^* = 1\%$ として、

- 目標0%なら、平時の名目金利は1%。**利下げの弾は1発分**しかない。
- 目標2%なら、平時の名目金利は3%。**弾は3発分**ある。

目標0%は一見『完璧な安定』だけど、
ちょっとした不況でZLBに激突して、
そこから先は4-1の自動引き締め装置──デフレで実質金利が勝手に上がる──が
起動する。0%目標は**床のすぐ上で綱渡りする設計**なんだお。
2%は、物価そのものの理想値というより、
**ZLBという断崖から距離を取るための安全マージン**だお。
第1幕でやらない夫が言った『恐怖から逆算された数字』の意味が分かったお。

**やらない夫**：
主因はそれだ。副次的な理由も2つ添えておけ。
第一に、消費者物価指数は品質向上や新商品への乗り換えを
捉えきれず、真のインフレを**過大評価する上方バイアス**を持つ。
計測値2%の実感は2%より低い。
第二に、名目賃金は下方に硬直的だ──賃下げは労使とも極端に嫌う。
ほどよいインフレは、名目賃下げなしに**実質賃金の調整**を可能にする
潤滑油（『のりしろ』）になる。
ゼロインフレの世界では、この調整弁が閉じる。

**やる夫**：
保留、全部回収したお。

保留その1：$V$ はなぜ潰れる → ZLBで貨幣と国債が同じ紙になるから（第2幕）。
保留その2：なぜ2% → ZLBからの安全マージン＋指数バイアス＋賃金ののりしろ（第6幕）。
保留その3：床の上で何ができる → 将来の金利経路の約束＝フォワードガイダンス（第5幕）。

**やらない夫**：
では、お前が歩いた道を振り返れ。

**やる夫**：
**『刷ればインフレ』の破綻**（$M$ 5倍、$V$ 5分の1）
→ **ZLBの発見**：金利ゼロで貨幣と国債が同じ紙になり、蛇口が空回りする
→ **視点の転換**：物価は資産価格。今日の物価は将来の政策予想が決める
→ **道具の構築**：$r = i - \pi^e$、見えない基準 $r^*$、
 $x = -\sigma(r-r^*)$、$\pi = \pi^e + \kappa x$ ──政策のループが閉じる
→ **候補の棄却**：裁量は信頼で死に、k%は $V$ で死に、
 1対1の金利対応はヴィクセルの棒で死に
→ **ほぼ一意の到達点**：インフレ目標＋テイラー原理＋フォワードガイダンス
→ **歴史での検証**：ボルカー、日本の四半世紀、世界インフレ

振り返ると、やる夫が引っかかった場所が全部要所だったお。
『$V$ は定数じゃない』『同じ紙の交換は効かない』
『名目1対1では実質が動かない』『$r^*$ は見えない』──
素朴な案がなぜだめかを一個ずつ潰したら、
現代の中央銀行がやってる形が、ほぼそのまま残ったお。

……でもやらない夫、最後に一つだけ嫌な予感を聞いてほしいお。

**やらない夫**：
言ってみろ。

**やる夫**：
4-2で予告された不吉な事実だお。**$r^*$ は下がり続けてる**。
高齢化で貯蓄（我慢の供給）が積み上がり、
成長鈍化で投資の実り（我慢の需要）が細れば、$r^*$ は沈む。
もし $r^*$ が0%やマイナスに定着したら、
目標2%でも平時の名目金利は2%かそれ以下──**弾はもう数発しかない**。
安全マージンの前提そのものが痩せていくお。
目標を3%、4%に上げるのかお？　でも上げた瞬間、
『目標は都合で動く』と思われて錨の信頼が壊れるお。
財政との役割分担も、中央銀行の独立の線引きも、全部問い直しになる……
この問題、答えはあるのかお？

**やらない夫**：
それは、まだ世界の誰も解いていない宿題だ──
次の総裁が向き合う問題、つまり明日からのお前の仕事だ。
今日はここまでにしよう。

## 付録　登場した道具の一覧

| 道具 | 式 | 役割 |
|---|---|---|
| 貨幣数量説（恒等式） | $MV = PY$ | $V$ 安定の仮定が死ぬと予言力を失う（第1幕） |
| フィッシャー方程式 | $r = i - \pi^e$ | 名目と実質の分離。予想が実体に入る入口（4-1） |
| 自然利子率 | $r^*$ | 緩和/引き締めを測る見えない基準点（4-2） |
| IS関係 | $x = -\sigma(r - r^*)$ | 実質金利（の経路）→需要（4-3） |
| フィリップス曲線 | $\pi = \pi^e + \kappa x$ | 需要→物価。粘着性ゆえ実体が調整を肩代わり（4-4） |
| テイラー・ルール | $i = r^* + \pi^{目標} + \phi(\pi - \pi^{目標}) + \phi_x x$ | $\phi > 1$（テイラー原理）が物価の錨（4-5） |
| 到達点 | インフレ目標＋テイラー原理＋フォワードガイダンス | 条件(1)〜(6)からほぼ一意（第5幕） |
