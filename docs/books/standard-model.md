# やる夫で学ぶ標準模型 ── 力を再発明する

---

## 登場人物

**やる夫** ── 特殊相対論と量子力学を学び、一般相対論を（やらない夫と）再発明し終えたばかりの学生。直観は鋭いが、すぐ安直な近道を試みて転ぶ。

**やらない夫** ── 物理学の教師。答えを教えず、反例と問いで導く。

---

## 第1幕　シュレーディンガーが壊れる日

---

**やる夫**：
「重力は再発明したお。時空の曲率、アインシュタイン方程式、ばっちりだお。
残りの力──電磁気力、弱い力、強い力──もさっさと片付けるお」

**やらない夫**：
「威勢がいいな。だが重力とは別の困難が待っている。
まず足元の確認だ。量子力学の基本方程式を書け」

**やる夫**：
「シュレーディンガー方程式だお」

$$i\hbar\frac{\partial \psi}{\partial t} = -\frac{\hbar^2}{2m}\nabla^2\psi + V\psi$$

**やらない夫**：
「この式はどこから来た？」

**やる夫**：
「エネルギーの式 $E = \dfrac{p^2}{2m} + V$ に、置き換え規則

$$E \to i\hbar\frac{\partial}{\partial t}, \qquad \mathbf{p} \to -i\hbar\nabla$$

を適用して、波動関数に作用させたものだお」

**やらない夫**：
「では聞く。$E = p^2/2m$ という出発点は、特殊相対論と整合するか？」

**やる夫**：
「……あっ。これは非相対論的な運動エネルギーだお。
式の見た目からしても、時間微分は1階なのに空間微分は2階。
時間と空間を対等に扱うローレンツ変換とは相容れない形だお。

電子を光速近くまで加速したら、この方程式は使えない。
原子の中の電子だって、内側の軌道では光速の数%で動いてるお。
**量子力学と特殊相対論が噛み合ってない**お」

**やらない夫**：
「そうだ。20世紀の二大理論は、生まれた瞬間から不仲だった。
直せるか？」

**やる夫**：
「簡単だお。出発点のエネルギーの式を相対論的なものに差し替えればいいお。
相対論的なエネルギーと運動量の関係は

$$E^2 = p^2c^2 + m^2c^4$$

これに同じ置き換え規則を適用するお。$E^2$ だから時間微分は2階になって」

$$-\hbar^2\frac{\partial^2 \psi}{\partial t^2} = -\hbar^2 c^2\nabla^2\psi + m^2c^4\psi$$

「整理すると」

$$\left(\frac{1}{c^2}\frac{\partial^2}{\partial t^2} - \nabla^2 + \frac{m^2c^2}{\hbar^2}\right)\psi = 0$$

「時間も空間も2階微分で対等。ローレンツ不変な形だお。
シンプルイズベストだお。これで相対論的量子力学は完成だお」

**やらない夫**：
「その方程式は **クライン・ゴルドン方程式** と呼ばれている。
では検収だ。量子力学で波動関数の絶対の要は何だった？」

**やる夫**：
「確率解釈だお。$\rho = |\psi|^2$ が位置の確率密度で、
全空間で積分すると1。時間が経っても1のまま──確率の保存だお」

**やらない夫**：
「シュレーディンガー方程式では、なぜ $|\psi|^2$ の全積分が時間変化しないんだった？」

**やる夫**：
「連続の方程式が成り立つからだお。
$\dfrac{\partial \rho}{\partial t} + \nabla\cdot\mathbf{j} = 0$ の形が方程式から導けて、
密度の減った分は流れ $\mathbf{j}$ が運び出した分、と勘定が合うお」

**やらない夫**：
「ではお前の新方程式で、連続の方程式を満たす $\rho$ を構成しろ。
やり方はシュレーディンガーのときと同じだ。
$\psi^\ast\times(\text{方程式}) - \psi\times(\text{方程式})^\ast$ を作れ」

**やる夫**：
「やってみるお。質量項は差し引きで消えて……
$\psi^\ast\partial_t^2\psi - \psi\partial_t^2\psi^\ast = \partial_t(\psi^\ast\partial_t\psi - \psi\partial_t\psi^\ast)$ を使うと、
連続の方程式の形になるお。時間微分の下に入るのは」

$$\rho = \frac{i\hbar}{2mc^2}\left(\psi^\ast\frac{\partial \psi}{\partial t} - \psi\frac{\partial \psi^\ast}{\partial t}\right)$$

「（係数は非相対論との接続のために調整したお）
……あれ？　$|\psi|^2$ じゃないお。$\psi$ の時間微分が入ってるお」

**やらない夫**：
「時間2階の方程式だから当然だ。初期条件として $\psi$ と $\partial_t\psi$ を独立に与えられる。
では判定してもらおう。一番簡単な解、平面波

$$\psi = Ne^{-i(Et - \mathbf{p}\cdot\mathbf{x})/\hbar}$$

を $\rho$ に代入しろ」

**やる夫**：
「$\partial_t\psi = -\dfrac{iE}{\hbar}\psi$ だから」

$$\rho = \frac{i\hbar}{2mc^2}\left(-\frac{iE}{\hbar} - \frac{iE}{\hbar}\right)|N|^2 = \frac{E}{mc^2}|N|^2$$

「密度が $E$ に比例するお。まあ非相対論極限では $E \approx mc^2$ だから
$\rho \approx |N|^2 = |\psi|^2$ に戻る。ちゃんと接続してるお。問題ないお」

**やらない夫**：
「本当か？　$E^2 = p^2c^2 + m^2c^4$ を $E$ について解け」

**やる夫**：
「$E = \pm\sqrt{p^2c^2 + m^2c^4}$ ……

**マイナスがあるお**。
2階の方程式だから、$E$ が負の解も正真正銘の解だお。
そして負エネルギー解を $\rho$ に入れると──

**確率密度が負になるお！！**
『そこに粒子が確率 −30% で存在する』なんて、日本語として壊れてるお！」

**やらない夫**：
「これがクライン・ゴルドン方程式の病気その1、負の確率だ」

**やる夫**：
「なら話は簡単だお。負エネルギー解は物理的に無意味として **捨てればいい** お。
方程式の解が全部物理的とは限らないお。昔からよくやる手だお」

**やらない夫**：
「捨てられない。理由は2つ。
第一に、正エネルギーの平面波だけでは数学的に完全系にならない。
任意の初期波束を展開しようとすると、負エネルギー成分が必要になる。
第二に、こちらが致命傷だが──仮に最初は正エネルギーだけの状態を用意しても、
外場との相互作用を入れた瞬間、負エネルギー状態への遷移が起きる。
量子力学では、禁止されていない遷移は必ず起きる。
しかも負のエネルギーには底がない。$E = -\infty$ まで際限なく落ちていく」

**やる夫**：
「原子が安定に存在できない理論になっちゃうお……。ぐぬぬ。
やる夫のシンプル案、完全に死んだお」

**やらない夫**：
「慰めておくと、この案を最初に試したのはシュレーディンガー本人だ。
1925年末、彼は有名な非相対論的方程式より **先に** 相対論版を書き下ろしている。
だが水素原子に適用すると微細構造の実験値と合わず、失望してお蔵入りにし、
非相対論版に後退してから発表した。
お前は歴史上最初の相対論的量子力学の、歴史通りの死に方を再現したわけだ」

**やる夫**：
「本人が先に転んでたのかお……。ちょっと安心したお。

でも敗因は分析できるお。
負エネルギーも負確率も、元をたどれば $E^2$ の **2乗** が悪いお。
2乗するから $\pm$ の両方が入り込む。時間微分が2階だから $\rho$ に $\partial_t\psi$ が入って正定値じゃなくなる。
だったら──**時間1階の方程式を作ればいい**。
$E = +\sqrt{p^2c^2 + m^2c^4}$ と、最初から正の平方根だけ取って量子化するお」

**やらない夫**：
「では、その平方根の演算子

$$\sqrt{-\hbar^2c^2\nabla^2 + m^2c^4}$$

の意味を言え。$\nabla^2$ が中に入った平方根とは何だ」

**やる夫**：
「うっ。……テイラー展開すればいいお。
$\sqrt{m^2c^4 + p^2c^2} = mc^2\sqrt{1 + p^2/m^2c^2}$ を展開すると
$p^2$、$p^4$、$p^6$……の無限級数。つまり $\nabla^2, \nabla^4, \nabla^6, \ldots$

**無限階の微分演算子** になるお。
……これはまずいお。ある点での微分を無限階まで全部知るってことは、
実質、関数の **全空間の情報** をその1点に集めることだお。
一点の物理が全宇宙に依存する。局所性が壊れて、相対論の因果律とも相性最悪だお」

**やらない夫**：
「そうだ。平方根を級数で処理する道は非局所性で死ぬ。
ここでディラックが1928年にやった発想の転換がこれだ：
**平方根の中身が、うまく完全平方になっていればいい**。

$$E = c\boldsymbol{\alpha}\cdot\mathbf{p} + \beta mc^2$$

と、$p$ の **1次式** で書けたと仮定する。係数 $\alpha_x, \alpha_y, \alpha_z, \beta$ に
何を要求すればいいか、両辺を2乗して調べろ」

**やる夫**：
「2乗して $E^2 = p^2c^2 + m^2c^4$ と一致すればいいんだお。
右辺を2乗すると、$\alpha$ 同士や $\alpha$ と $\beta$ の積の順序に気をつけて展開して……
交差項が全部消えて、2乗の項の係数が1になる条件は」

$$\alpha_i^2 = \beta^2 = 1, \qquad \alpha_i\alpha_j + \alpha_j\alpha_i = 0 \ (i \neq j), \qquad \alpha_i\beta + \beta\alpha_i = 0$$

「ふつうの数で解くお。えーと、$\alpha_x\alpha_y + \alpha_y\alpha_x = 2\alpha_x\alpha_y = 0$ だから
$\alpha_x$ か $\alpha_y$ がゼロ。でも $\alpha_i^2 = 1$ だからどっちもゼロじゃない──

**矛盾だお**。ふつうの数では絶対に解けないお」

**やらない夫**：
「なぜ矛盾した？」

**やる夫**：
「$\alpha_x\alpha_y + \alpha_y\alpha_x = 2\alpha_x\alpha_y$ とまとめた瞬間だお。
掛け算の順序を入れ替えられると仮定した。
つまり──**順序を入れ替えられないもの** なら望みがあるお。**行列だお！**

反交換する行列なら知ってるお。スピンで習ったパウリ行列だお。
$\sigma_x\sigma_y + \sigma_y\sigma_x = 0$、$\sigma_i^2 = 1$。条件にぴったりだお。
$\alpha_i = \sigma_i$ とすれば2×2で済むお」

**やらない夫**：
「$\beta$ はどうする。$\sigma_x, \sigma_y, \sigma_z$ 全部と反交換する4つ目の行列が要るぞ」

**やる夫**：
「2×2のエルミート行列は $1, \sigma_x, \sigma_y, \sigma_z$ の4つで展開できるから、
$\beta = a_0 1 + \mathbf{a}\cdot\boldsymbol{\sigma}$ と書いて反交換条件を課すと……
$\{\beta, \sigma_x\} = 0$ から $a_0 = a_x = 0$、同様に全成分ゼロ。$\beta = 0$。
でも $\beta^2 = 1$ に反する。

**2×2では席が足りないお**。パウリ行列は3つが限界で、4つ目が座れないお」

**やらない夫**：
「3×3も駄目なことが示せる（奇数次元では条件を満たせない）。最小は **4×4** だ。
したがって波動関数 $\psi$ は4成分の縦ベクトルになる。
こうして得られるのが **ディラック方程式** だ」

$$i\hbar\frac{\partial\psi}{\partial t} = \left(c\boldsymbol{\alpha}\cdot(-i\hbar\nabla) + \beta mc^2\right)\psi$$

「時間1階。そして確率密度は $\rho = \psi^\dagger\psi = \sum_a |\psi_a|^2$、明白に正定値だ。
負確率の病気は完治した。
おまけに、要求していなかった賞品が2つ転がり出る。
4成分の内訳を調べると、電子の **スピン** が自動的に含まれており、
磁気モーメントの値 $g = 2$ まで正しく出る。
スピンは手で付け足す飾りではなく、相対論と量子力学の整合が強制する構造だった」

**やる夫**：
「すごいお！　1階化しただけでスピンがおまけについてきたお！
……ん？　4成分？　スピンの上下で2成分のはずだお。残りの2成分は何だお。

……嫌な予感がするお。エネルギー固有値を出すお。
平面波を入れて4×4行列の固有値問題を解くと、固有値は……

$E = +\sqrt{p^2c^2+m^2c^4}$ が2つ（スピン上下）と、
$E = -\sqrt{p^2c^2+m^2c^4}$ が **2つ**。

**負エネルギー解がまだいるお！！**
時間1階にしたのに！　確率は正になったのに、エネルギーの底抜けは治ってないお！
じゃあ電子はやっぱり $E=-\infty$ に向かって落ち続けるお。この理論も死ぬお」

**やらない夫**：
「ディラックも同じ崖に立った。そして1930年、開き直りとしか言いようのない手を打つ。
電子はフェルミ粒子だ。パウリの排他原理──同じ状態には1個しか入れない。
そこでディラックはこう宣言した：
**真空とは、負エネルギー状態がすべて電子で埋め尽くされた状態である**」

**やる夫**：
「はあ！？　真空が電子で満杯！？
……いや、待つお。理屈は通ってるのかお……？
負の席が全部埋まってるなら、正エネルギーの電子は排他原理で **落ちる先がない**。
だから安定。原子は壊れない。

むちゃくちゃな絵だけど、筋は通ってるお……。
あ、でも面白いことが起きるお。
満杯の海に十分なエネルギー（$2mc^2$ 以上）の光を当てたら、
海の中の電子を1個、正エネルギーに叩き上げられるお。
すると海に **空席（穴）** が残る。
満杯が『何もない』に見えるなら、穴は『何かある』に見えるはずだお。
電荷 $-e$ の欠落だから、穴は電荷 **$+e$** を持った粒子に見える……

**プラスの電荷を持った、電子と同じ質量の粒子！？　そんなもの見たことないお！**」

**やらない夫**：
「1932年、アンダーソンが宇宙線の霧箱写真の中にそれを見つけた。
**陽電子** ──史上初の反物質だ。
お前がいま予言した『光が電子と穴のペアを作る』過程は **対生成**、
電子が穴に落ちて光を出して両方消える過程は **対消滅** として、どちらも観測されている。
負エネルギー解という理論の欠陥が、反物質の予言に化けた。
物理学史でも屈指の大逆転だ」

**やる夫**：
「失敗作の穴が新粒子だったのかお……。しびれるお。
じゃあディラックの海で一件落着だお。相対論的量子力学、完成だお！」

**やらない夫**：
「残念だが、海は応急処置だ。もっと深い病巣を見せる。
思考実験だ。電子を1個、箱に閉じ込めて、箱をどんどん小さくしろ。
不確定性関係から何が起きる？」

**やる夫**：
「箱のサイズ $\Delta x$ を絞ると、運動量の不確定さが $\Delta p \sim \hbar/\Delta x$ で増えるお。
するとエネルギーの不確定さも増える。相対論的には $\Delta E \sim c\Delta p$ だから

$$\Delta E \sim \frac{\hbar c}{\Delta x}$$

……で、それが何だお？　非相対論の量子力学でもいつもやってる話だお」

**やらない夫**：
「非相対論では起きえないことが、相対論では起きる。
$E = mc^2$ ──エネルギーは質量に転換できる。
$\Delta E$ が電子・陽電子ペアの生成閾値 $2m_ec^2$ を超えるのは、$\Delta x$ がいくらのときだ」

**やる夫**：
「$\dfrac{\hbar c}{\Delta x} \sim 2m_ec^2$ を解いて（係数2は気にしないお）$\Delta x \sim \dfrac{\hbar}{m_ec} \approx 4\times10^{-13}\mathrm{m}$……

箱を **0.4ピコメートル**──水素原子の百分の一ほど、電子のコンプトン波長のスケール──まで絞ると、
閉じ込めに使ったエネルギー自体が **電子・陽電子ペアを生んでしまう** お。

……ちょっと待つお。これ、根本的にまずいお。
『電子1個の位置を精密に測る』つもりが、測ろうとした瞬間に電子が2個3個に増える。
箱の中の電子が『さっきの1個』なのか『いま湧いた1個』なのか、原理的に区別できない。

**『1粒子の波動関数 $\psi(\mathbf{x})$』という考え方そのものが、相対論と両立しないお！！**
負エネルギーとか負確率とかの個別の病気じゃなくて、
『粒子の個数は決まっている』という **枠組みの前提** が崩れてるんだお」

**やらない夫**：
「それが真の病巣だ。相対論は粒子の生成・消滅を許す。
ならば理論の基本変数は『N個の粒子の位置』ではありえない。
粒子数が変わっても存在し続ける、より根源的な何かでなければならない。
心当たりは？」

**やる夫**：
「……あるお。**電磁場** だお。
光子は生まれたり消えたりするけど、電磁場そのものは常に全空間にあるお。
光子は『電磁場の励起』──場という海の、さざ波の一粒だお。

そうか、これを全部の粒子に広げるんだお。
電子にも『電子場』があって、電子はその励起。
場が基本で、粒子は場の量子。
粒子が生まれるのは場が励起されること、消えるのは励起が収まること。
個数が変わっても場は困らないお」

**やらない夫**：
「それが **場の量子論** だ。
クライン・ゴルドン方程式もディラック方程式も、
『1粒子の波動方程式』としては死ぬが、『場の方程式』として完全復活する
（実際、この読み替えでクライン・ゴルドン方程式の負確率問題は消える──第2幕で見せる）。

これで舞台は決まった：**特殊相対論と整合する、場の量子論**。
だが舞台が決まっただけだ。台本がない。
電子場と電磁場はどう相互作用する？　弱い力の場、強い力の場はどんな形をしている？
無数に書けるはずの理論の中から、自然が採用した台本を選び出す **原理** は何か。

その原理は、お前が『当たり前すぎて無内容』と切り捨てそうな場所に隠れている」

## 第2幕　誰も見ていない回転 ── 対称性と保存則

---

**やらない夫**：
「本題の前に、記法の大掃除をする。$c$ と $\hbar$ をいちいち書くのは以後つらい。
**自然単位系**：$\hbar = c = 1$ と置く」

**やる夫**：
「そんな勝手が許されるのかお。$c$ は秒速30万kmだお」

**やらない夫**：
「単位の選び方の問題にすぎない。
『1秒』の代わりに『光が1メートル進む時間』を時間の単位に選べば $c = 1$ だ。
結果、エネルギー・質量・運動量が全部同じ単位（GeVが標準）になり、
長さと時間はその逆数になる。換算率だけ覚えておけ：$\hbar c \approx 197\mathrm{MeV\cdot fm}$。
最後に実験と比べるときに $\hbar$ と $c$ を復元すればいい」

**やる夫**：
「じゃあ電子の質量 0.5 MeV の逆数が電子のコンプトン波長……
$197\mathrm{MeV\cdot fm} / 0.5\mathrm{MeV} \approx 390\mathrm{fm}$。
第1幕で出した $4\times10^{-13}\mathrm{m}$ と一致だお。換算機として使えてるお」

**やらない夫**：
「よし。では本題だ。
量子力学で、波動関数 $\psi$ と、それを位相回転した $e^{i\alpha}\psi$（$\alpha$ は実定数）。
観測で区別できるか？」

**やる夫**：
「できないお。観測にかかるのは $|\psi|^2$ とか $\psi^\ast\hat{O}\psi$ の形の量で、
$e^{i\alpha}$ は $e^{-i\alpha}$ と必ずペアで現れて相殺するお。
全体位相は物理じゃない。量子力学の初日に習うことだお。

……で、それが何だお？
『変えても何も変わらないものがある』って、文字通り **何も言ってない** お。

**やらない夫**：

**やらない夫**：

いま一度、変換をよく見ろ」

$$\phi(x) \to e^{i\alpha}\phi(x), \qquad \alpha = \text{定数}$$

「この $\alpha$ が **定数** だという意味を、宇宙のスケールで想像してみろ」

## 第3幕　「一斉に」への疑い

---

**やる夫**：

…………ん？

んん？？　ちょっと待つお。
『全宇宙で一斉に』って、**誰がやるんだお**？

ここ地球で位相を 30° 回すとき、アンドロメダ銀河の場の位相も
同時に 30° 回さないと対称変換にならない。
でも特殊相対論を思い出すお。『同時』は観測者依存で、

どこにあるんだお？」

**やらない夫**：
「続けろ」

**やる夫**：
「一般相対論のときと同じ匂いがするお。
あのとき、座標系は『場所ごとに勝手に選んでいい、物理はその選択に依存するな』
と要請したお。時空全体を覆う特権的な座標系なんて認めなかった。

$$\phi(x) \to e^{i\alpha(x)}\phi(x)$$

**やらない夫**：
「定数 $\alpha$ の変換を **大域的（グローバル）変換**、

では現状の理論が局所変換で不変か、判定してもらおう。
場の理論の基本部品は $\phi$ とその微分 $\partial_\mu\phi$ だ。
局所変換のもとで $\partial_\mu\phi$ がどう変換するか、計算しろ。1行で済む」

**やる夫**：

「……**余計なのが出たお**。

微分には $i(\partial_\mu\alpha)\phi$ という寄生虫がくっつくお。
$\alpha$ が定数なら $\partial_\mu\alpha = 0$ で消えるけど、局所変換では消えない。

これだと、たとえば運動項 $\partial_\mu\phi^\ast\partial^\mu\phi$ は不変じゃなくなる。
**微分が入った瞬間、局所対称性は壊れる** お」

**やらない夫**：
「そうだ。分かれ道だ。選べ。

**道A**：『局所対称性は成り立っていない。対称性は大域的なもので十分』と認める。
現状の理論のままで、何も困らない。安全な道だ。

**道B**：『局所対称性は成り立つ **べき** だ』と要請し、

**やる夫**：

道Bで行くお。
『位相の基準は場所ごとに自由。物理はその選択に依存しない』を **設計原理に格上げ** するお。

**やらない夫**：
「その要請を **ゲージ原理** と呼ぶ。
ただし言っておくが、道Bはタダでは通れない。
いまお前自身が示した通り、素の微分 $\partial_\mu$ は局所変換で壊れる。
壊れないようにするには、理論に何かを **足す** 必要がある」

**やる夫**：
「寄生虫 $i(\partial_\mu\alpha)\phi$ を打ち消す係の『何か』だお。

でも、そんな都合のいい場が、注文通りに自然界に存在するのかお？

**やらない夫**：

第2幕で先送りにした『保存流の製造機』──ラグランジアン形式とネーターの定理の一般形だ。

## 第4幕　道具の構築 ── 対称性を担ぐ数学

---

### 4-1　場のラグランジアン ── 対称性が見える形式

---

**やらない夫**：
「解析力学は履修済みだな。点粒子では、ラグランジアン $L(q, \dot q)$ と
最小作用の原理からオイラー・ラグランジュ方程式が出た」

$$\frac{d}{dt}\frac{\partial L}{\partial \dot q} - \frac{\partial L}{\partial q} = 0$$

**やる夫**：
「正直に言うと、あれの有り難みが今ひとつ分かってないお。
運動方程式さえあれば物理は全部計算できるお。
わざわざ $L$ を経由するのは回り道じゃないかお。
クライン・ゴルドン方程式だってもう手元にあるお」

**やらない夫**：
「では試験だ。クライン・ゴルドン方程式

$$(\Box + m^2)\phi = 0$$

を眺めて、この理論が位相対称性を持つこと、ローレンツ不変であること、
そして **他にどんな対称性を持ちうるか** を読み取れ。方程式だけからだ」

**やる夫**：
「位相対称性は……$\phi \to e^{i\alpha}\phi$ を代入すれば確認できるお。
ローレンツ不変も $\Box$ の形から言えなくはない……
でも『他にどんな対称性があるか』を **網羅** しろと言われると、
方程式をひたすら睨んで当てずっぽうで試すしかないお。うぐぐ」

**やらない夫**：
「そこがラグランジアンの存在意義その1だ。
クライン・ゴルドン方程式を生むラグランジアン（正確にはラグランジアン密度）はこれだ」

$$\mathcal{L} = \partial_\mu\phi^\ast\partial^\mu\phi - m^2\phi^\ast\phi$$

「対称性の判定は『$\mathcal{L}$ が不変かどうか』の一発で済む。
$\phi \to e^{i\alpha}\phi$ なら、どの項でも $\phi^\ast$ と $\phi$ がペアだから
$e^{-i\alpha}e^{i\alpha} = 1$ で **見た瞬間に不変** と分かる。
ローレンツ不変性も、$\mathcal{L}$ が添字の縮約されたスカラーである時点で自動保証。
対称性を扱う理論では、$\mathcal{L}$ は回り道ではなく本街道だ。
存在意義その2は次節で見せる──ネーターの定理の製造工程が $\mathcal{L}$ を通る。

ではこの $\mathcal{L}$ が本当にクライン・ゴルドン方程式を吐くか、検算しろ。
場のオイラー・ラグランジュ方程式は、点粒子の $t$ 微分が時空微分に広がった形だ」

$$\partial_\mu\frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^\ast)} - \frac{\partial\mathcal{L}}{\partial\phi^\ast} = 0$$

**やる夫**：
「$\phi$ と $\phi^\ast$ は独立扱いでいいんだったお（実部・虚部の組み替えだから）。
$\phi^\ast$ で変分すると、$\dfrac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^\ast)} = \partial^\mu\phi$、
$\dfrac{\partial\mathcal{L}}{\partial\phi^\ast} = -m^2\phi$ だから」

$$\partial_\mu\partial^\mu\phi + m^2\phi = (\Box + m^2)\phi = 0$$

「出たお。第1幕の方程式そのものだお。
$\phi$ 側で変分すれば複素共役の方程式が出て、これも整合してるお」

**やらない夫**：
「以後、理論を指定するとは $\mathcal{L}$ を指定することだと思え。
部品表を渡しておく。スピン0の複素場は今の $\mathcal{L}$。
スピン1/2のディラック場は

$$\mathcal{L} = \bar\psi\left(i\gamma^\mu\partial_\mu - m\right)\psi$$

（$\gamma^\mu$ は第1幕の $\alpha, \beta$ を相対論的に整理し直した4×4行列、
$\bar\psi = \psi^\dagger\gamma^0$。変分するとディラック方程式が出る──後で使う）」

### 4-2　保存流の製造機 ── ネーターの定理

---

**やらない夫**：
「第2幕の借りを返す。ネーターの定理の一般形を組み立てろ。
設定：$\mathcal{L}(\phi, \partial_\mu\phi)$ が、無限小変換

$$\phi \to \phi + \epsilon\Delta\phi$$

（$\epsilon$ は無限小の定数パラメータ）で不変、つまり $\delta\mathcal{L} = 0$ とする。
このとき $\delta\mathcal{L}$ を愚直に展開しろ」

**やる夫**：
「連鎖律で」

$$\delta\mathcal{L} = \frac{\partial\mathcal{L}}{\partial\phi}\epsilon\Delta\phi + \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi)}\epsilon\partial_\mu(\Delta\phi)$$

**やらない夫**：
「第1項に運動方程式を使え。いま考えている場は実際の運動の解だ」

**やる夫**：
「オイラー・ラグランジュから $\dfrac{\partial\mathcal{L}}{\partial\phi} = \partial_\mu\dfrac{\partial\mathcal{L}}{\partial(\partial_\mu\phi)}$。代入すると全体が積の微分にまとまるお」

$$\delta\mathcal{L} = \epsilon\partial_\mu\left(\frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi)}\Delta\phi\right)$$

「で、対称性の仮定から $\delta\mathcal{L} = 0$。ということは括弧の中身を

$$j^\mu \equiv \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi)}\Delta\phi$$

と名付ければ $\partial_\mu j^\mu = 0$。

**保存流が機械的に出てきたお**。
入力は『$\mathcal{L}$』と『変換のかたち $\Delta\phi$』の2つだけ。
対称性を放り込むと保存則が出てくる自動販売機だお」

**やらない夫**：
「それがネーターの定理だ（場が複数あれば各場の寄与を足す）。
では販売機の検収だ。複素スカラー場の位相変換で電流を製造しろ。
$e^{i\alpha} \approx 1 + i\alpha$ だから $\Delta\phi = i\phi$、$\Delta\phi^\ast = -i\phi^\ast$ だ」

**やる夫**：
「$\phi$ と $\phi^\ast$ の寄与を足して」

$$j^\mu = \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi)}(i\phi) + \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^\ast)}(-i\phi^\ast) = i\left(\partial^\mu\phi^\ast\cdot\phi - \partial^\mu\phi\cdot\phi^\ast\right)$$

「符号を整理すると $j^\mu = -i(\phi^\ast\partial^\mu\phi - \phi\partial^\mu\phi^\ast)$……
**第2幕の電流と一致だお**（全体符号は電荷の正負の約束の違いだけ）。
あのとき天下りで検算した式が、製造機から正規品として出荷されたお。

実スカラー場の不合格判定もやり直せるお。
実場には位相変換 $\Delta\phi = i\phi$ がそもそも定義できない（実数条件を壊す）から、
販売機に入れる硬貨がない。電流なし。第2幕の結論と同じだお」

**やらない夫**：
「付記しておく。$\Delta\phi$ に時空並進を入れればエネルギー・運動量テンソル $T^{\mu\nu}$ が出る。
一般相対論でお前が場の方程式の右辺に据えた、あの $T^{\mu\nu}$ の出生地はここだ。
工具は揃った。問1──微分の修理に取りかかる」

### 4-3　共変微分の再発明 ── そして打ち消し係の正体

---

**やらない夫**：
「現状確認。局所変換 $\phi \to e^{i\alpha(x)}\phi$ のもとで

$$\partial_\mu\phi \to e^{i\alpha}\Big(\partial_\mu\phi + i(\partial_\mu\alpha)\phi\Big)$$

寄生虫 $i(\partial_\mu\alpha)\phi$ をどう始末する」

**やる夫**：
「一番安直な案から行くお。**悪い項を手で引けばいい** お。
新しい微分を $D_\mu\phi \equiv \partial_\mu\phi - i(\partial_\mu\alpha)\phi$ と定義するお」

**やらない夫**：
「では聞く。その定義に出てくる $\alpha(x)$ とは、どの $\alpha$ だ」

**やる夫**：
「どのって、変換に使った……

……あっ。だめだお。
$\alpha$ は『これから誰かが行うかもしれない変換』のパラメータであって、
**理論の中に存在する量じゃない** お。
変換する前の理論に、変換のパラメータを部品として組み込むなんて論理が循環してるお。
微分の定義は、$\alpha$ を一切使わずに書けてないといけないお」

**やらない夫**：
「良い自己添削だ。では問題の根っこに戻ろう。
そもそも微分とは何をする操作だ。定義に立ち返れ」

**やる夫**：
「隣り合う2点での場の値を比べる操作だお。

### 4-4　場の強さ ── 内部空間の曲率

スピンの上下のアナロジーで **アイソスピン** と呼ぶ。2成分をまとめて書くと」

$$\psi = \begin{pmatrix}\psi_p \\ \psi_n\end{pmatrix}$$

「この2成分の空間での『回転』──陽子と中性子を混ぜ合わせる変換

$$\psi \to U\psi, \qquad U = \exp\left(i\alpha^a\frac{\sigma^a}{2}\right) \quad (a = 1,2,3)$$

（$\sigma^a$ はパウリ行列、$\alpha^a$ は3つの回転角）で強い力が不変、というのが彼の観察だ。
この行列群を **SU(2)** と呼ぶ。
1954年、ヤンとミルズが問うた：**この対称性を局所化したらどうなるか**」

**やる夫**：
「やる夫たちが位相でやったことの完コピをすればいいお。
$\alpha^a$ を $\alpha^a(x)$ にして、共変微分を作る。
打ち消し係のゲージ場も、回転角が3つあるんだから3本必要で、行列にまとめて

$$A_\mu = A_\mu^a\frac{\sigma^a}{2}, \qquad D_\mu = \partial_\mu - igA_\mu$$

場の強さも完コピで $F_{\mu\nu} = \partial_\mu A_\nu - \partial_\nu A_\mu$。以上、終わりだお。
U(1)でさんざん鍛えたんだから、記号を差し替えるだけだお」

**やらない夫**：
「では判定してもらおう。
まず $A_\mu$ の変換則から。U(1) のときと同じ要求──$D_\mu\psi \to U D_\mu\psi$──から逆算すると、
今回はこうなる（導出は同じ手順だ、$U$ が行列になっただけ）」

$$A_\mu \to U A_\mu U^\dagger - \frac{i}{g}(\partial_\mu U)U^\dagger$$

**やる夫**：
「まず退化検算をさせてほしいお。$U$ が1×1、つまり $U = e^{i\alpha}$ のただの数なら、
$UA_\mu U^\dagger = A_\mu$（数は可換）、$-\frac{i}{g}(\partial_\mu U)U^\dagger = -\frac{i}{g}i(\partial_\mu\alpha) = \frac{1}{g}\partial_\mu\alpha$。
合わせて $A_\mu + \frac{1}{g}\partial_\mu\alpha$──**4-3の変換則にぴったり戻る** お。上位互換の形だお。

で、行列の場合を見比べると……
第2項が $\partial_\mu\alpha$ の対応物なのは分かるお。
でも第1項、U(1) なら $UA_\mu U^\dagger = e^{i\alpha}A_\mu e^{-i\alpha} = A_\mu$ で消えてた回転が、
行列だと $UA_\mu U^\dagger \neq A_\mu$ で **残る** お。
$A_\mu$ 自身も回されるんだお。……嫌な予感がしてきたお」

**やらない夫**：
「その予感を計算で確かめろ。お前の完コピ案 $F_{\mu\nu} = \partial_\mu A_\nu - \partial_\nu A_\mu$ は
ゲージ変換でどうなる。
不変（$F \to F$）は行列では望めないから、要求を一段緩めておく：
共変（$F \to UFU^\dagger$）なら合格だ。トレースを取れば不変量が作れるからな。
簡単のため $U$ が定数行列の場合──大域変換──だけでいいから判定しろ」

**やる夫**：
「$U$ 定数なら $\partial_\mu U = 0$ で、変換は $A_\mu \to UA_\mu U^\dagger$ だけ。すると

$$F^\prime_{\mu\nu} = \partial_\mu(UA_\nu U^\dagger) - \partial_\nu(UA_\mu U^\dagger) = U(\partial_\mu A_\nu - \partial_\nu A_\mu)U^\dagger = UF_{\mu\nu}U^\dagger$$

あれ、合格に見えるお。……いや待つお、これは大域変換に限った話だお。
$U$ が場所依存だと $\partial_\mu U$ の項が両方から出て、
U(1) では $\partial_\mu\partial_\nu\alpha$ の対称性で消えてくれたのに、
行列版では $(\partial_\mu U)U^\dagger$ と $A$ の **積の項** が残る。
行列は掛ける順序が効くから、たすき掛けで相殺してた項が相殺しなくなるお。

……原因は言語化できたお。**非可換性** だお。
U(1) の計算のあちこちで『掛け算の順序を入れ替えて相殺』を無意識に使ってた。
行列ではそれが全部使えない。完コピは死んだお。ぐぬぬ」

**やらない夫**：
「では正しい $F$ をどう見つける。お前はもう、座標に依らない道具の作り方を2回学んでいる」

**やる夫**：
「交換子だお。定義に戻るお。$[D_\mu, D_\nu]\psi$ を行列のまま計算するお。
$D_\mu = \partial_\mu - igA_\mu$ で、今度は $A_\mu A_\nu \neq A_\nu A_\mu$ に気をつけて……

U(1) のときは $A_\mu A_\nu$ の項が $\mu\leftrightarrow\nu$ の引き算で消えたけど、
今回は消えずに交換子 $[A_\mu, A_\nu]$ が生き残って」

$$[D_\mu, D_\nu]\psi = -ig\left(\partial_\mu A_\nu - \partial_\nu A_\mu - ig[A_\mu, A_\nu]\right)\psi$$

「つまり本物の場の強さは」

$$F_{\mu\nu} = \partial_\mu A_\nu - \partial_\nu A_\mu - ig[A_\mu, A_\nu]$$

**やらない夫**：
「これなら $F_{\mu\nu} \to UF_{\mu\nu}U^\dagger$ が局所変換でも成り立つ（$[D_\mu,D_\nu]$ の変換性から自動だ）。
合格・不合格の両検算をしておけ」

**やる夫**：
「合格例：U(1)、つまり1×1行列の場合。数同士は可換だから $[A_\mu, A_\nu] = 0$ で、
4-4の $F_{\mu\nu} = \partial_\mu A_\nu - \partial_\nu A_\mu$ に **ぴったり退化** するお。
U(1) は非可換版の特別な場合として吸収された。上位互換だお。

不合格例の確認：交換子項を落とした完コピ版は、さっき見たとおり局所変換で共変にならない。
両方向とも期待どおりだお。

で、運動項は $F$ の2乗のトレース（トレースは $UFU^\dagger$ の $U$ を巡回で消すため）」

$$\mathcal{L}_A = -\frac{1}{2} \mathrm{tr}  \left(F_{\mu\nu} F^{\mu\nu} \right)$$

「……ちょっと待つお。この中身、ヤバいものが入ってるお。
$F = \partial A - \partial A - ig[A, A]$ の2乗だから、展開すると
$(\partial A)^2$ だけじゃなくて、$g(\partial A)A^2$ の **3次の項** と $g^2 A^4$ の **4次の項** が出るお。

場の3次・4次の項って、相互作用項だお。つまり──
**ゲージ場が、ゲージ場自身と相互作用する**！？」

**やらない夫**：
「それが非可換ゲージ理論（**ヤン・ミルズ理論**）の最大の個性だ。
物理的に言えばこうだ。
U(1) の光子は、電荷を『運ぶ』が自分は電荷を『持たない』。だから光同士はすれ違うだけ。
非可換のゲージ場は、変換則 $A \to UAU^\dagger + \cdots$ が示すとおり **自分自身が回される**──
つまり自分自身が『荷』を持つ。荷を持つ以上、ゲージ場同士が力を及ぼし合う」

**やる夫**：
「郵便屋さん自身が手紙を持ってるようなものかお。
……そしてまた対応表だお。**重力も同じ** だったお。
重力場自身がエネルギーを持ち、エネルギーは重力の源だから、重力場は自分自身と相互作用する。
アインシュタイン方程式が非線形で解くのが地獄だったのは、それが理由だったお。
ヤン・ミルズも同じ理由で非線形。
U(1) だけが例外的に『郵便屋が手ぶら』の線形理論だったんだお」

| | 一般相対論 | U(1)ゲージ理論 | 非可換ゲージ理論 |
|---|---|---|---|

**やる夫**：

**やる夫**：

**やらない夫**：

**やる夫**：
「断面積は面積の次元、自然単位系で $\mathrm{GeV}^{-2}$。
振幅が $G_F$ に比例するから断面積は $G_F^2$ に比例して、次元は $\mathrm{GeV}^{-4}$。

破綻するエネルギーは $\sigma \sim G_F^2E^2$ が上限に達するところで、だいたい

$$E \sim \frac{1}{\sqrt{G_F}} \approx 300\ \mathrm{GeV}$$

300 GeV 付近で、フェルミ理論は **確率が1を超える**。理論の自壊だお」

**やらない夫**：

桁の見積もりで **100 GeV 級** だお。陽子の100倍。
