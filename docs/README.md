# genai-textbooks

Short Textbooks on Various Topics Written by Generative AI

生成 AI が執筆した、さまざまなトピックの短い教科書集です。

どのトピックも、「なんとなくの知識はあるが具体的にどういう理論が登場するのかもう少しリアルな内容が知りたい」という興味に答える目的で作成しています。

内容の専門的な正しさは担保されていないため、その分野の具体的な内容に興味を持つための導入教材という位置づけでご利用ください。

コンテンツの内容は品質の向上のため改訂されることがあります。

## 教材一覧

### 社会

#### [やる夫が日銀総裁になるようです](/books/economics-japan-bank)
問い：お金を刷れば物価は上がるのか、中央銀行は何を操作して景気とインフレに働きかけるのか。
プロット：日銀総裁に任命されたやる夫が、貨幣数量説の素朴な理解から出発し、金利・期待・国債市場・金融政策の限界を再発見していく。

#### [やる夫が政令指定都市の市長になるようです](/books/local-government)
問い：市長はどこまで都市を動かせるのか、自治体の予算・議会・行政組織はどのように噛み合うのか。
プロット：新人市長のやる夫が、給食費無償化などの公約を実行しようとして、二元代表制、財政制約、行政実務の壁にぶつかる。

#### [やる夫が文部科学大臣になったようです](/books/education)
問い：教育は予算や制度を増やせば良くなるのか、学力や格差を左右する要因は何か。
プロット：文部科学大臣になったやる夫が、思いつきの改革案を歴史的な実験や統計の反例で崩されながら、教育政策の難しさを学ぶ。

#### [やる夫が世界平和を目指すようです](/books/world-peace)
問い：平和とは単に戦争がないことなのか、国際社会は紛争をどのように予防・管理できるのか。
プロット：国連事務総長になったやる夫が、理想的な平和案を冷戦・抑止・安全保障制度の現実に照らして作り直していく。

#### [やる夫が夢見るエネルギー政策](/books/energy-policy)
問い：安さ、安定供給、脱炭素、安全保障を同時に満たすエネルギーミックスはどう設計できるのか。
プロット：エネルギー政策の責任者になったやる夫が、単一目標への全振り案を次々に失敗させ、電源構成のトレードオフを掘り下げる。

#### [やる夫と牛丼と食料政策](/books/japan-food-policy)
問い：食料自給率を上げれば本当に安全なのか、日本の食料安全保障は何で成り立っているのか。
プロット：深夜の牛丼屋で一杯の牛丼を分解しながら、やる夫が輸入、備蓄、価格、農業政策のつながりをたどる。

#### [やる夫が悟りを開きそうです](/books/buddhism)
問い：「空」や「悟り」は、単にすべてが無意味だということなのか。
プロット：般若心経を分かった気になったやる夫が、敗北への執着を手がかりに、空・縁起・苦の見方を問い直していく。

#### [やる夫が時をかけて哲学するようです](/books/philosophy)
問い：「当たり前」を疑うことには何の意味があり、哲学はどのように常識を作り替えてきたのか。
プロット：やる夫が古代から現代までの時代を渡り歩き、各時代の常識に閉じ込められながら哲学上の問いを追体験する。

### 技術

#### [やる夫が高層ビルを建てるそうです](/books/skyscraper)
問い：高層ビルは、重力・風・地震にどう耐えて空へ伸びるのか。
プロット：240mのビルを建てようとするやる夫が、組積造から鉄骨、耐震・制振設計へと、人類の失敗の階段を踏み直す。

#### [やる夫が携帯電話会社のCTOになるそうです](/books/mobile)
問い：声やデータはどのように電波へ載り、限られた周波数で多数の利用者をつなぐのか。
プロット：携帯電話会社のCTOになったやる夫が、アンテナ、変調、セル設計、シャノン限界の壁に順番にぶつかる。

#### [やる夫が天気予報を作るそうです](/books/weather-forcast)
問い：天気予報はなぜ難しく、観測・物理モデル・データ同化・AIは何を補っているのか。
プロット：明日の天気を当てたいだけのやる夫が、持続予報やアナログ予報の限界から数値予報の仕組みへ進んでいく。

#### [やる夫で学ぶ電力系統](/books/power-grid)
問い：電気を在庫できない社会で、需要と供給、周波数、安定度をどう保つのか。
プロット：電力会社CTOになったやる夫が、離島グリッドの小さな例をいじりながら、発電機・送電網・需給調整を再設計する。

#### [やる夫で学ぶデータベース設計](/books/database-design)
問い：正規化だけで業務システムは作れるのか、データの意味・履歴・制約はどう設計するのか。
プロット：給与計算システムを題材に、やる夫がExcel風の表を崩しながら、実務に耐えるテーブル設計を組み立てる。

#### [やる夫が作る総合感冒薬](/books/cold-medicine)
問い：風邪薬は風邪を治しているのか、一錠の中身はどの症状をどう抑えているのか。
プロット：風邪を一発で治す薬を求めるやる夫が、対症療法、成分の役割、副作用と配合設計を薬局で学ぶ。

#### [やる夫が新型電池を量産するそうです](/books/chemical-plant)
問い：フラスコで一度成功した反応を、そのまま巨大工場で再現できないのはなぜか──その反応、工場になるのか。
プロット：新型電池の固体電解質を発見して舞い上がる院生やる夫が、スケールアップ・分離・材料・立地・時間・安全の壁に一つずつ転びながら、化学工場が「化学」ではなく巨大システム工学であることを再発見する。

#### [やる夫が圏論チョットワカルかもしれません](/books/fp-category-theory)
問い：圏論やモナドは、プログラミングの何を抽象化しているのか。
プロット：関数型プログラミングで挫折したやる夫が、日々書くコードを足場に、射・関手・自然変換・モナドへ近づく。

### 科学・数学

#### [やる夫で学ぶ一般相対性理論](/books/general-relativity)
問い：ニュートン重力の瞬時作用を捨てると、重力は何として記述されるべきなのか。
プロット：特殊相対論を学んだやる夫が、遅延する力の失敗から時空の曲率とアインシュタイン方程式へ導かれる。

#### [やる夫で学ぶ標準模型](/books/standard-model)
問い：量子力学と特殊相対論を両立させると、粒子と力はどのように見えてくるのか。
プロット：シュレーディンガー方程式の限界を出発点に、やる夫が場、対称性、ゲージ理論、ヒッグス機構へ進む。

#### [やる夫で学ぶ軌道力学](/books/orbital-mechanics-moon)
問い：月へ行って帰るには、なぜ一直線ではなく軌道を設計しなければならないのか。
プロット：月旅行を思い立ったやる夫が、大砲案や単純加速案を退けられ、Δv、遷移軌道、帰還の計算を積み上げる。

#### [やる夫で学ぶシュレーディンガー方程式](/books/schrodinger)
問い：古典物理が破綻したあと、波動関数とシュレーディンガー方程式は何を説明するために必要になるのか。
プロット：黒体放射や水素スペクトルの謎から出発し、やる夫がボーア模型の限界を越えて量子力学の形を再発見する。

#### [やる夫で学ぶ数学基礎論](/books/mathematical-foundations)
問い：「1+1=2」はなぜ正しく、数学は集合・論理・証明の上にどう立っているのか。
プロット：数学動画の断片知識に惹かれたやる夫が、自然数、集合、形式体系、不完全性へと根本の問いを追っていく。
